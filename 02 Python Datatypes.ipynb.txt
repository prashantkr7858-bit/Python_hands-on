    "## Python Built-in Numeric Data Types\n",
    "\n",
    "Python offers three distinct built-in numeric types:\n",
    "\n",
    "1.  **`int` (Integer):** Represents whole numbers (positive, negative, or zero) without a fractional part. In Python 3, integers have arbitrary precision, meaning they can grow as large as your system's memory allows.\n",
    "2.  **`float` (Floating-Point Number):** Represents real numbers, including those with a fractional part indicated by a decimal point or scientific notation (e.g., `1.5e3` for 1500.0). Floats typically use the IEEE 754 standard for double-precision representation, which can sometimes lead to small precision inaccuracies.\n",
    "3.  **`complex` (Complex Number):** Represents numbers with a real and an imaginary part. The imaginary part is denoted by a `j` or `J` suffix (e.g., `3+5j`). While less common in general data metrics, they are crucial in many scientific and engineering domains.\n",

    "### d) Floor Division (`//`)\n",
    "**Explanation:** Divides the first number by the second and rounds the result *down* to the nearest whole number (integer).\n",
    "**Examples:**\n",
    "```python\n",
    "# 1. Calculating number of full batches\n",
    "total_students = 157\n",
    "students_per_batch = 50\n",
    "full_batches = total_students // students_per_batch\n",
    "print(f\"1. Number of full batches: {full_batches}\") # Output: 3\n",
    "\n",
    "# 2. Distributing assignments equally\n",
    "total_assignments = 47\n",
    "num_instructors = 4\n",
    "assignments_per_instructor = total_assignments // num_instructors\n",
    "print(f\"2. Assignments per instructor (min): {assignments_per_instructor}\") # Output: 11\n",
    "\n",
    "# 3. Calculating completed weeks\n",
    "days_elapsed = 30\n",
    "days_per_week = 7\n",
    "completed_weeks = days_elapsed // days_per_week\n",
    "print(f\"3. Completed weeks: {completed_weeks}\") # Output: 4\n",
    "\n",
    "# 4. Dividing by a larger number\n",
    "small_group = 10\n",
    "large_capacity = 100\n",
    "result = small_group // large_capacity\n",
    "print(f\"4. Result of 10 // 100: {result}\") # Output: 0\n",
    "\n",
    "# 5. Floor division with zero numerator\n",
    "zero_students = 0\n",
    "batch_size = 50\n",
    "result = zero_students // batch_size\n",
    "print(f\"5. Result of 0 // 50: {result}\") # Output: 0\n",
    "\n",
    "# 6. Floor division with negative numbers (rounds down)\n",
    "debt = -100\n",
    "payments_per_month = 30\n",
    "months_of_full_payments = debt // payments_per_month # How many full payment periods fit into the debt? Needs care with sign.\n",
    "print(f\"6. Result of -100 // 30: {months_of_full_payments}\") # Output: -4\n",
    "\n",
    "# 7. Another negative example\n",
    "val1 = -10\n",
    "val2 = 3\n",
    "result = val1 // val2\n",
    "print(f\"7. Result of -10 // 3: {result}\") # Output: -4\n",
    "\n",
    "# 8. Positive numerator, negative denominator\n",
    "val1 = 10\n",
    "val2 = -3\n",
    "result = val1 // val2\n",
    "print(f\"8. Result of 10 // -3: {result}\") # Output: -4\n",
    "\n",
    "# 9. Calculating full sets of resources\n",
    "total_licenses = 125\n",
    "licenses_per_set = 10\n",
    "full_sets = total_licenses // licenses_per_set\n",
    "print(f\"9. Full sets of licenses: {full_sets}\") # Output: 12\n",
    "\n",
    "# 10. Integer part of an average (if calculated stepwise)\n",
    "total_score = 485\n",
    "num_students = 50\n",
    "integer_part_avg = total_score // num_students\n",
    "print(f\"10. Integer part of average score: {integer_part_avg}\") # Output: 9\n",
    "```\n",

    "### e) Modulo (`%`)\n",
    "**Explanation:** Returns the *remainder* of the division of the first number by the second.\n",
    "**Examples:**\n",
    "```python\n",
    "# 1. Finding remaining students after forming full batches\n",
    "total_students = 157\n",
    "students_per_batch = 50\n",
    "remaining_students = total_students % students_per_batch\n",
    "print(f\"1. Students for the last (incomplete) batch: {remaining_students}\") # Output: 7\n",
    "\n",
    "# 2. Finding leftover assignments\n",
    "total_assignments = 47\n",
    "num_instructors = 4\n",
    "leftover_assignments = total_assignments % num_instructors\n",
    "print(f\"2. Leftover assignments to distribute: {leftover_assignments}\") # Output: 3\n",
    "\n",
    "# 3. Finding remaining days in the current week\n",
    "days_elapsed = 30\n",
    "days_per_week = 7\n",
    "day_in_week = days_elapsed % days_per_week # (Assuming day 0 is the start)\n",
    "print(f\"3. Day number within the current week (0-6): {day_in_week}\") # Output: 2\n",
    "\n",
    "# 4. Checking for even or odd numbers\n",
    "student_id = 1501\n",
    "is_odd = student_id % 2\n",
    "print(f\"4. Is student ID {student_id} odd (1 means yes)? {is_odd}\") # Output: 1\n",
    "batch_id = 102\n",
    "is_even = batch_id % 2\n",
    "print(f\"   Is batch ID {batch_id} even (0 means yes)? {is_even}\") # Output: 0\n",
    "\n",
    "# 5. Remainder when dividing by a larger number\n",
    "small_group = 10\n",
    "large_capacity = 100\n",
    "remainder = small_group % large_capacity\n",
    "print(f\"5. Remainder of 10 % 100: {remainder}\") # Output: 10\n",
    "\n",
    "# 6. Modulo with zero numerator\n",
    "zero_students = 0\n",
    "batch_size = 50\n",
    "remainder = zero_students % batch_size\n",
    "print(f\"6. Remainder of 0 % 50: {remainder}\") # Output: 0\n",
    "\n",
    "# 7. Modulo with negative numerator (result takes sign of divisor)\n",
    "debt_remaining = -7\n",
    "monthly_payment = 30\n",
    "result = debt_remaining % monthly_payment\n",
    "print(f\"7. Result of -7 % 30: {result}\") # Output: 23 (because -7 = -1*30 + 23)\n",
    "\n",
    "# 8. Another negative example\n",
    "val1 = -10\n",
    "val2 = 3\n",
    "remainder = val1 % val2\n",
    "print(f\"8. Result of -10 % 3: {remainder}\") # Output: 2 (because -10 = -4*3 + 2)\n",
    "\n",
    "# 9. Positive numerator, negative denominator\n",
    "val1 = 10\n",
    "val2 = -3\n",
    "remainder = val1 % val2\n",
    "print(f\"9. Result of 10 % -3: {remainder}\") # Output: -2 (because 10 = -4*-3 + (-2))\n",
    "\n",
    "# 10. Finding leftover licenses\n",
    "total_licenses = 125\n",
    "licenses_per_set = 10\n",
    "leftover_licenses = total_licenses % licenses_per_set\n",
    "print(f\"10. Leftover licenses: {leftover_licenses}\") # Output: 5\n",
    "```\n",

    "### f) Exponentiation (`**`)\n",
    "**Explanation:** Raises the first number to the power of the second number.\n",
    "**Examples:**\n",
    "```python\n",
    "# 1. Modeling simplified exponential student growth (base * factor^periods)\n",
    "initial_students = 100\n",
    "growth_factor = 2\n",
    "periods = 3\n",
    "projected_students = initial_students * (growth_factor ** periods)\n",
    "print(f\"1. Projected students after {periods} periods: {projected_students}\") # Output: 800\n",
    "\n",
    "# 2. Calculating squares (power of 2)\n",
    "side_length = 5\n",
    "area = side_length ** 2\n",
    "print(f\"2. Area of square: {area}\") # Output: 25\n",
    "\n",
    "# 3. Calculating cubes (power of 3)\n",
    "edge_length = 3\n",
    "volume = edge_length ** 3\n",
    "print(f\"3. Volume of cube: {volume}\") # Output: 27\n",
    "\n",
    "# 4. Power of zero (always 1, except 0**0 is debated but 1 in Python)\n",
    "base = 10\n",
    "result = base ** 0\n",
    "print(f\"4. 10 to the power of 0: {result}\") # Output: 1\n",
    "zero_base = 0\n",
    "result_zero = zero_base ** 0\n",
    "print(f\"   0 to the power of 0: {result_zero}\") # Output: 1\n",
    "\n",
    "# 5. Power of one (always the base itself)\n",
    "base = total_students # 1500\n",
    "result = base ** 1\n",
    "print(f\"5. {base} to the power of 1: {result}\") # Output: 1500\n",
    "\n",
    "# 6. Large powers (possible due to arbitrary precision)\n",
    "large_power = 2 ** 100\n",
    "print(f\"6. 2 to the power of 100 (first few digits): {str(large_power)[:10]}...\") # Output: 1267650600...\n",
    "\n",
    "# 7. Negative base, even power (positive result)\n",
    "base = -5\n",
    "power = 2\n",
    "result = base ** power\n",
    "print(f\"7. (-5) to the power of 2: {result}\") # Output: 25\n",
    "\n",
    "# 8. Negative base, odd power (negative result)\n",
    "base = -5\n",
    "power = 3\n",
    "result = base ** power\n",
    "print(f\"8. (-5) to the power of 3: {result}\") # Output: -125\n",
    "\n",
    "# 9. Using exponentiation for bit shifting (2**n is like 1 << n)\n",
    "bit_shift_equivalent = 2 ** 3\n",
    "print(f\"9. 2 to the power of 3: {bit_shift_equivalent}\") # Output: 8 (same as 1 << 3)\n",
    "\n",
    "# 10. Calculating compound interest (simplified, int only)\n",
    "principal = 10000\n",
    "rate_percent = 5 # Integer percent\n",
    "periods = 3\n",
    "# Formula P * (1 + r)^n -> Needs float usually, but simplified:\n",
    "# This is NOT accurate finance, just showing the operator\n",
    "approx_value = principal * (100 + rate_percent) ** periods // (100 ** periods)\n",
    "print(f\"10. Simplified compound interest approx: {approx_value}\") # Output: 11576 (approximate)\n",
    "```\n",

    "### h) `pow(base, exp[, mod])`\n",
    "**Explanation:** A built-in function alternative to `**`. `pow(x, y)` is equivalent to `x ** y`. It also supports an optional third argument for efficient modular exponentiation `(x ** y) % z`.\n",
    "**Examples:**\n",
    "```python\n",
    "# 1. Simple power calculation\n",
    "result = pow(10, 3)\n",
    "print(f\"1. pow(10, 3): {result}\") # Output: 1000\n",
    "\n",
    "# 2. Equivalent to growth example above\n",
    "initial_students = 100\n",
    "growth_factor = 2\n",
    "periods = 3\n",
    "projected_students = initial_students * pow(growth_factor, periods)\n",
    "print(f\"2. Projected students using pow(): {projected_students}\") # Output: 800\n",
    "\n",
    "# 3. Power of 2\n",
    "side_length = 5\n",
    "area = pow(side_length, 2)\n",
    "print(f\"3. Area using pow(): {area}\") # Output: 25\n",
    "\n",
    "# 4. Power of 0\n",
    "base = total_students # 1500\n",
    "result = pow(base, 0)\n",
    "print(f\"4. pow({base}, 0): {result}\") # Output: 1\n",
    "\n",
    "# 5. Power of 1\n",
    "base = 42\n",
    "result = pow(base, 1)\n",
    "print(f\"5. pow(42, 1): {result}\") # Output: 42\n",
    "\n",
    "# 6. Negative base, even power\n",
    "result = pow(-5, 2)\n",
    "print(f\"6. pow(-5, 2): {result}\") # Output: 25\n",
    "\n",
    "# 7. Negative base, odd power\n",
    "result = pow(-5, 3)\n",
    "print(f\"7. pow(-5, 3): {result}\") # Output: -125\n",
    "\n",
    "# 8. Modular exponentiation: (3 ** 4) % 5\n",
    "result_mod = pow(3, 4, 5) # 3^4 = 81. 81 % 5 = 1\n",
    "print(f\"8. pow(3, 4, 5): {result_mod}\") # Output: 1\n",
    "\n",
    "# 9. Modular exponentiation: (2 ** 10) % 7\n",
    "result_mod_large = pow(2, 10, 7) # 2^10 = 1024. 1024 % 7 = 2\n",
    "print(f\"9. pow(2, 10, 7): {result_mod_large}\") # Output: 2\n",
    "\n",
    "# 10. Modular exponentiation with negative base (handle carefully based on definition)\n",
    "# pow(-2, 3, 5) -> (-2)^3 = -8. (-8) % 5 -> Python gives 2 (-8 = -2*5 + 2)\n",
    "result_mod_neg = pow(-2, 3, 5)\n",
    "print(f\"10. pow(-2, 3, 5): {result_mod_neg}\") # Output: 2\n",
    "```\n",
    "\n",
    "### i) `divmod(a, b)`\n",
    "**Explanation:** Takes two numbers (a, b) and returns a pair of numbers (a tuple) consisting of their quotient (`a // b`) and remainder (`a % b`).\n",
    "**Examples:**\n",
    "```python\n",
    "# 1. Getting full batches and remaining students simultaneously\n",
    "total_students = 157\n",
    "students_per_batch = 50\n",
    "batches, remaining = divmod(total_students, students_per_batch)\n",
    "print(f\"1. Batches: {batches}, Remaining Students: {remaining}\") # Output: Batches: 3, Remaining Students: 7\n",
    "\n",
    "# 2. Distributing assignments and finding leftovers\n",
    "total_assignments = 47\n",
    "num_instructors = 4\n",
    "assign_per, leftover_assign = divmod(total_assignments, num_instructors)\n",
    "print(f\"2. Assignments per instructor: {assign_per}, Leftovers: {leftover_assign}\") # Output: Assignments per instructor: 11, Leftovers: 3\n",
    "\n",
    "# 3. Calculating weeks and remaining days\n",
    "days_elapsed = 30\n",
    "days_per_week = 7\n",
    "weeks, rem_days = divmod(days_elapsed, days_per_week)\n",
    "print(f\"3. Weeks: {weeks}, Remaining Days: {rem_days}\") # Output: Weeks: 4, Remaining Days: 2\n",
    "\n",
    "# 4. Dividing by a larger number\n",
    "small_group = 10\n",
    "large_capacity = 100\n",
    "quotient, remainder = divmod(small_group, large_capacity)\n",
    "print(f\"4. divmod(10, 100): Quotient={quotient}, Remainder={remainder}\") # Output: Quotient=0, Remainder=10\n",
    "\n",
    "# 5. Dividing zero\n",
    "zero_students = 0\n",
    "batch_size = 50\n",
    "quotient, remainder = divmod(zero_students, batch_size)\n",
    "print(f\"5. divmod(0, 50): Quotient={quotient}, Remainder={remainder}\") # Output: Quotient=0, Remainder=0\n",
    "\n",
    "# 6. Negative numerator\n",
    "debt = -100\n",
    "payments_per_month = 30\n",
    "q, r = divmod(debt, payments_per_month)\n",
    "print(f\"6. divmod(-100, 30): Quotient={q}, Remainder={r}\") # Output: Quotient=-4, Remainder=20\n",
    "\n",
    "# 7. Another negative numerator example\n",
    "q, r = divmod(-10, 3)\n",
    "print(f\"7. divmod(-10, 3): Quotient={q}, Remainder={r}\") # Output: Quotient=-4, Remainder=2\n",
    "\n",
    "# 8. Positive numerator, negative denominator\n",
    "q, r = divmod(10, -3)\n",
    "print(f\"8. divmod(10, -3): Quotient={q}, Remainder={r}\") # Output: Quotient=-4, Remainder=-2\n",
    "\n",
    "# 9. Calculating full sets and leftovers\n",
    "total_licenses = 125\n",
    "licenses_per_set = 10\n",
    "sets, leftover_licenses = divmod(total_licenses, licenses_per_set)\n",
    "print(f\"9. Full sets: {sets}, Leftover licenses: {leftover_licenses}\") # Output: Full sets: 12, Leftover licenses: 5\n",
    "\n",
    "# 10. Converting total minutes to hours and minutes\n",
    "total_minutes = 135\n",
    "minutes_per_hour = 60\n",
    "hours, minutes = divmod(total_minutes, minutes_per_hour)\n",
    "print(f\"10. {total_minutes} mins = {hours} hours and {minutes} minutes\") # Output: 135 mins = 2 hours and 15 minutes\n",
    "```\n",

    "## 2. `float` (Floating-Point Number)\n",
    "\n",
    "**Explanation:** Floats represent numbers with a decimal point. They are essential for measurements, averages, percentages, financial calculations, and any quantity that might not be a whole number, like `average_rating`. Be mindful of potential small precision errors inherent in binary floating-point representation.\n",
    "\n",
    "**Common Operations & Functions/Methods:**\n",
    "\n",
    "Floats support the same arithmetic operators (`+`, `-`, `*`, `/`, `//`, `%`, `**`) as integers, but the results are typically floats (especially with `/`). They also work with `abs()`, `pow()`, `divmod()`, and have specific methods.\n",
    "\n",
    "### a) Arithmetic Operations (`+`, `-`, `*`, `/`)\n",
    "**Explanation:** Standard arithmetic. Division (`/`) always results in a float, even if the mathematical result is a whole number.\n",
    "**Examples (showing `/` specifically):**\n",
    "```python\n",
    "# 1. Calculating average score precisely\n",
    "total_score = 485.0 # Use float for precision\n",
    "num_students = 50\n",
    "average_score = total_score / num_students\n",
    "print(f\"1. Precise average score: {average_score}\") # Output: 9.7\n",
    "\n",
    "# 2. Calculating completion percentage\n",
    "completed_modules = 7.0\n",
    "total_modules = 15\n",
    "completion_rate = (completed_modules / total_modules) * 100\n",
    "print(f\"2. Course completion rate: {completion_rate:.2f}%\") # Output: 46.67% (formatted)\n",
    "\n",
    "# 3. Dividing integers results in float\n",
    "revenue = 750000\n",
    "total_students = 1500\n",
    "revenue_per_student = revenue / total_students\n",
    "print(f\"3. Revenue per student: {revenue_per_student}\") # Output: 500.0 (Note the .0)\n",
    "\n",
    "# 4. Simple division\n",
    "val1 = 10.0\n",
    "val2 = 4.0\n",
    "result = val1 / val2\n",
    "print(f\"4. 10.0 / 4.0 = {result}\") # Output: 2.5\n",
    "\n",
    "# 5. Division involving zero\n",
    "result = 0.0 / 5.0\n",
    "print(f\"5. 0.0 / 5.0 = {result}\") # Output: 0.0\n",
    "\n",
    "# 6. Division by small number\n",
    "value = 1.0\n",
    "divisor = 0.001\n",
    "result = value / divisor\n",
    "print(f\"6. 1.0 / 0.001 = {result}\") # Output: 1000.0\n",
    "\n",
    "# 7. Using division in scaling\n",
    "original_rating = 4.8\n",
    "scale_factor = 2.0 # Example: Adjusting scale\n",
    "scaled_rating = original_rating / scale_factor\n",
    "print(f\"7. Scaled rating: {scaled_rating}\") # Output: 2.4\n",
    "\n",
    "# 8. Negative division\n",
    "val1 = -15.0\n",
    "val2 = 2.0\n",
    "result = val1 / val2\n",
    "print(f\"8. -15.0 / 2.0 = {result}\") # Output: -7.5\n",
    "\n",
    "# 9. Division resulting in repeating decimal (shows precision limits)\n",
    "result = 1.0 / 3.0\n",
    "print(f\"9. 1.0 / 3.0 = {result}\") # Output: 0.3333333333333333\n",
    "\n",
    "# 10. Calculating average monthly growth\n",
    "total_growth = 55.5 # e.g. average increase over period\n",
    "months = 6\n",
    "avg_monthly = total_growth / months\n",
    "print(f\"10. Average monthly growth: {avg_monthly}\") # Output: 9.25\n",
    "```\n",
    "*(Examples for `+`, `-`, `*` are similar to `int` but operate on/produce floats)*\n",
    "\n",
    "### b) Floor Division (`//`) and Modulo (`%`) on Floats\n",
    "**Explanation:** These operators also work with floats, following the same logic (floor division rounds down, modulo gives remainder), but the results can be floats. `divmod()` also works.\n",
    "**Examples (using `//`):**\n",
    "```python\n",
    "# 1. How many full 'rating points' in average_rating\n",
    "average_rating = 4.8\n",
    "full_points = average_rating // 1.0\n",
    "print(f\"1. Full rating points in {average_rating}: {full_points}\") # Output: 4.0\n",
    "\n",
    "# 2. Number of full $10 units in a price\n",
    "price = 49.95\n",
    "units_of_10 = price // 10.0\n",
    "print(f\"2. Full $10 units in {price}: {units_of_10}\") # Output: 4.0\n",
    "\n",
    "# 3. Dividing floats with floor division\n",
    "val1 = 10.5\n",
    "val2 = 2.1\n",
    "result = val1 // val2\n",
    "print(f\"3. 10.5 // 2.1 = {result}\") # Output: 5.0\n",
    "\n",
    "# 4. Negative float floor division\n",
    "val1 = -10.5\n",
    "val2 = 2.1\n",
    "result = val1 // val2\n",
    "print(f\"4. -10.5 // 2.1 = {result}\") # Output: -6.0 (rounds down towards negative infinity)\n",
    "\n",
    "# 5. Using very small divisor\n",
    "val1 = 1.0\n",
    "val2 = 0.3\n",
    "result = val1 // val2\n",
    "print(f\"5. 1.0 // 0.3 = {result}\") # Output: 3.0\n",
    "\n",
    "# 6. Floor division by 1.0\n",
    "val = 7.8\n",
    "result = val // 1.0\n",
    "print(f\"6. 7.8 // 1.0 = {result}\") # Output: 7.0\n",
    "\n",
    "# 7. Floor division result is zero\n",
    "val1 = 2.5\n",
    "val2 = 3.0\n",
    "result = val1 // val2\n",
    "print(f\"7. 2.5 // 3.0 = {result}\") # Output: 0.0\n",
    "\n",
    "# 8. Calculating full weeks from fractional days\n",
    "days = 30.5\n",
    "days_per_week = 7.0\n",
    "full_weeks = days // days_per_week\n",
    "print(f\"8. Full weeks in {days} days: {full_weeks}\") # Output: 4.0\n",
    "\n",
    "# 9. Large numbers\n",
    "val1 = 1000.0\n",
    "val2 = 150.5\n",
    "result = val1 // val2\n",
    "print(f\"9. 1000.0 // 150.5 = {result}\") # Output: 6.0\n",
    "\n",
    "# 10. Floor division involving integers (promotes to float)\n",
    "result = 10 // 3.0\n",
    "print(f\"10. 10 // 3.0 = {result}\") # Output: 3.0\n",
    "```\n",
    "*(`%` and `divmod` examples follow similarly)*\n",
    "\n",
    "### c) `round(number[, ndigits])`\n",
    "**Explanation:** Rounds a number to a specified number of decimal places (`ndigits`). If `ndigits` is omitted or `None`, it rounds to the nearest integer (returns an `int`). Uses \"round half to even\" strategy for values exactly halfway between two numbers (e.g., `round(2.5)` is 2, `round(3.5)` is 4).\n",
    "**Examples:**\n",
    "```python\n",
    "# 1. Rounding the average rating to one decimal place\n",
    "average_rating = 4.8356\n",
    "rounded_rating_1dp = round(average_rating, 1)\n",
    "print(f\"1. Average rating rounded to 1dp: {rounded_rating_1dp}\") # Output: 4.8\n",
    "\n",
    "# 2. Rounding the average rating to the nearest integer\n",
    "rounded_rating_int = round(average_rating)\n",
    "print(f\"2. Average rating rounded to nearest int: {rounded_rating_int}\") # Output: 5\n",
    "\n",
    "# 3. Rounding completion rate to two decimal places\n",
    "completion_rate = 46.6666666\n",
    "rounded_rate_2dp = round(completion_rate, 2)\n",
    "print(f\"3. Completion rate rounded to 2dp: {rounded_rate_2dp}\") # Output: 46.67\n",
    "\n",
    "# 4. Rounding to zero decimal places (returns float)\n",
    "price = 49.95\n",
    "rounded_price_0dp = round(price, 0)\n",
    "print(f\"4. Price rounded to 0dp: {rounded_price_0dp}\") # Output: 50.0\n",
    "\n",
    "# 5. Round half to even: 2.5\n",
    "result = round(2.5)\n",
    "print(f\"5. round(2.5): {result}\") # Output: 2\n",
    "\n",
    "# 6. Round half to even: 3.5\n",
    "result = round(3.5)\n",
    "print(f\"6. round(3.5): {result}\") # Output: 4\n",
    "\n",
    "# 7. Rounding a negative number\n",
    "neg_value = -4.8356\n",
    "rounded_neg_1dp = round(neg_value, 1)\n",
    "print(f\"7. -4.8356 rounded to 1dp: {rounded_neg_1dp}\") # Output: -4.8\n",
    "\n",
    "# 8. Rounding to nearest integer (negative)\n",
    "rounded_neg_int = round(neg_value)\n",
    "print(f\"8. -4.8356 rounded to nearest int: {rounded_neg_int}\") # Output: -5\n",
    "\n",
    "# 9. Rounding to negative digits (rounds to tens, hundreds...)\n",
    "total_students_precise = 1537.5\n",
    "rounded_to_tens = round(total_students_precise, -1)\n",
    "print(f\"9. Students rounded to nearest 10: {rounded_to_tens}\") # Output: 1540\n",
    "\n",
    "# 10. Rounding to nearest hundred\n",
    "revenue = 753450.75\n",
    "rounded_revenue_hundreds = round(revenue, -2)\n",
    "print(f\"10. Revenue rounded to nearest 100: {rounded_revenue_hundreds}\") # Output: 753500\n",
    "```\n",
    "\n",
    "### d) `float.is_integer()`\n",

    "print(f\"2. Is {revenue_per_student} an integer? {revenue_per_student.is_integer()}\") # Output: True\n",
    "\n",
    "# 3. Check zero\n",
    "zero_float = 0.0\n",
    "print(f\"3. Is {zero_float} an integer? {zero_float.is_integer()}\") # Output: True\n",
    "\n",
    "# 4. Check a negative integer float\n",
    "neg_int_float = -10.0\n",
    "print(f\"4. Is {neg_int_float} an integer? {neg_int_float.is_integer()}\") # Output: True\n",
    "\n",
    "# 5. Check a negative non-integer float\n",
    "neg_non_int_float = -10.5\n",
    "print(f\"5. Is {neg_non_int_float} an integer? {neg_non_int_float.is_integer()}\") # Output: False\n",

    "# 6. Result of a calculation\n",
    "result = 10.0 / 2.0 # 5.0\n",

    "# 7. Result of another calculation\n",
    "result = 10.0 / 3.0 # 3.333...\n",

    "# 9. Very small fractional part (may be False due to precision)\n",

    "# 10. A float resulting from rounding\n",

    "\n",
    "### e) `float.as_integer_ratio()`\n",
    "**Explanation:** Method called on a float object. Returns a pair (tuple) of two integers whose ratio is exactly equal to the float value. Useful for representing floats precisely as fractions. The fraction is always in simplest form (lowest terms).\n",

    "```python\n",

    "# 2. Ratio for average rating (shows potential complexity)\n",
    "average_rating = 4.8\n",
    "ratio = average_rating.as_integer_ratio()\n",
    "# 4.8 = 48/10 = 24/5\n",
    "print(f\"2. Ratio for {average_rating}: {ratio}\") # May show large numbers due to precision: (5404319552844595, 1125899906842624) -> this IS 4.8\n",
    "# Let's try with a simpler representation if possible\n",
    "ratio_simple = (4.75).as_integer_ratio() # 4.75 = 19/4\n",
    "print(f\"   Ratio for 4.75: {ratio_simple}\") # Output: (19, 4)\n",
    "\n",
    "# 3. Ratio for an integer float\n",
    "ratio = (500.0).as_integer_ratio()\n",
    "print(f\"3. Ratio for 500.0: {ratio}\") # Output: (500, 1)\n",
    "\n",
    "# 4. Ratio for 0.25\n",
    "ratio = (0.25).as_integer_ratio()\n",
    "print(f\"4. Ratio for 0.25: {ratio}\") # Output: (1, 4)\n",
    "\n",
    "# 5. Ratio for a negative float\n",
    "ratio = (-0.75).as_integer_ratio()\n",
    "print(f\"5. Ratio for -0.75: {ratio}\") # Output: (-3, 4)\n",
    "\n",
    "# 6. Ratio for zero\n",
    "ratio = (0.0).as_integer_ratio()\n",
    "print(f\"6. Ratio for 0.0: {ratio}\") # Output: (0, 1)\n",
    "\n",
    "# 7. Ratio demonstrating precision limits (0.1 is not exact in binary)\n",
    "ratio = (0.1).as_integer_ratio()\n",
    "print(f\"7. Ratio for 0.1 (precision): {ratio}\") # Output: (3602879701896397, 36028797018963968) - NOT (1, 10)\n",
    "\n",
    "# 8. Ratio for 1.5\n",
    "ratio = (1.5).as_integer_ratio()\n",
    "print(f\"8. Ratio for 1.5: {ratio}\") # Output: (3, 2)\n",
    "\n",
    "# 9. Ratio for a large float\n",
    "large_float = 12345.6789\n",
    "ratio = large_float.as_integer_ratio()\n",
    "print(f\"9. Ratio for {large_float} (can be large): {ratio}\") # Output: (some large integers)\n",
    "\n",
    "# 10. Ratio from a calculation result\n",
    "result = 10.0 / 8.0 # 1.25\n",
    "ratio = result.as_integer_ratio()\n",
    "print(f\"10. Ratio for 1.25: {ratio}\") # Output: (5, 4)\n",
    "```\n",
    "\n",
    "### f) `float()`\n",
    "**Explanation:** The type constructor, can also be used as a function to convert other types (like `int` or compatible strings) to a float.\n",
    "**Examples:**\n",
    "```python\n",
    "# 1. Converting total students (int) to float\n",
    "total_students = 1500\n",
    "students_float = float(total_students)\n",
    "print(f\"1. Total students as float: {students_float}\") # Output: 1500.0\n",

    "## 3. `complex` (Complex Number)\n",
    "\n",
    "**Explanation:** Complex numbers have a real part and an imaginary part (suffixed with `j`). They are created using `complex(real, imag)` or by writing the literal form like `10+3j`. While your `growth_projection = complex(10, 3)` example is valid, applying complex numbers meaningfully to typical Edukron metrics like student count or ratings is abstract; they are more common in fields like electrical engineering, signal processing, quantum mechanics, and certain mathematical visualizations. We'll use the `growth_projection` variable and other examples.\n",
    "\n",
    "**Common Operations & Attributes/Methods:**\n",
    "\n",
    "Complex numbers support standard arithmetic operations (`+`, `-`, `*`, `/`, `**`). They also work with `abs()` (which calculates the magnitude), `pow()`, and have specific attributes `.real`, `.imag`, and a method `.conjugate()`.\n",
    "\n",
    "### a) Arithmetic Operations (`+`, `-`, `*`, `/`)\n",
    "**Explanation:** Standard arithmetic, following the rules of complex number math.\n",
    "**Examples (using `*`):**\n",
    "```python\n",
    "# Let's define two complex projections/factors\n",
    "growth_projection = complex(10, 3) # 10 + 3j\n",
    "adjustment_factor = complex(1, -0.5) # 1 - 0.5j\n",
    "\n",
    "# 1. Multiplying two complex numbers\n",
    "# (a+bj)(c+dj) = (ac-bd) + (ad+bc)j\n",
    "# (10+3j)(1-0.5j) = (10*1 - 3*(-0.5)) + (10*(-0.5) + 3*1)j\n",
    "#                 = (10 + 1.5) + (-5 + 3)j = 11.5 - 2j\n",
    "result = growth_projection * adjustment_factor\n",
    "print(f\"1. {growth_projection} * {adjustment_factor} = {result}\") # Output: (11.5-2j)\n",
    "\n",
    "# 2. Multiplying by a real number (int or float)\n",
    "scale_factor = 2\n",
    "scaled_projection = growth_projection * scale_factor\n",
    "print(f\"2. {growth_projection} * {scale_factor} = {scaled_projection}\") # Output: (20+6j)\n",
    "\n",
    "# 3. Multiplying by pure imaginary number (j)\n",
    "imag_factor = 1j\n",
    "rotated_projection = growth_projection * imag_factor # Rotates by 90 deg\n",
    "# (10+3j)*j = 10j + 3j^2 = 10j - 3 = -3 + 10j\n",
    "print(f\"3. {growth_projection} * {imag_factor} = {rotated_projection}\") # Output: (-3+10j)\n",
    "\n",
    "# 4. Squaring a complex number\n",
    "squared_projection = growth_projection ** 2\n",
    "# (10+3j)^2 = 100 + 2*10*3j + (3j)^2 = 100 + 60j - 9 = 91 + 60j\n",
    "print(f\"4. ({growth_projection})^2 = {squared_projection}\") # Output: (91+60j)\n",
    "\n",
    "# 5. Multiplying by zero complex number\n",
    "zero_complex = complex(0, 0)\n",
    "result = growth_projection * zero_complex\n",
    "print(f\"5. {growth_projection} * {zero_complex} = {result}\") # Output: 0j\n",
    "\n",
    "# 6. Multiplying conjugates: (a+bj)(a-bj) = a^2 + b^2 (a real number)\n",
    "conjugate = growth_projection.conjugate() # 10-3j\n",
    "result = growth_projection * conjugate\n",
    "# (10+3j)(10-3j) = 10^2 + 3^2 = 100 + 9 = 109\n",
    "print(f\"6. {growth_projection} * {conjugate} = {result}\") # Output: (109+0j)\n",
    "\n",
    "# 7. Multiplying complex numbers involving negatives\n",
    "c1 = complex(-2, 1) # -2 + j\n",
    "c2 = complex(1, -3) # 1 - 3j\n",
    "# (-2+j)(1-3j) = (-2*1 - 1*(-3)) + (-2*(-3) + 1*1)j\n",
    "#              = (-2 + 3) + (6 + 1)j = 1 + 7j\n",
    "result = c1 * c2\n",
    "print(f\"7. {c1} * {c2} = {result}\") # Output: (1+7j)\n",
    "\n",
    "# 8. Multiplication resulting in pure real\n",
    "c1 = 1+1j\n",
    "c2 = 1-1j\n",
    "result = c1 * c2 # (1+1j)(1-1j) = 1^2 + 1^2 = 2\n",
    "print(f\"8. {c1} * {c2} = {result}\") # Output: (2+0j)\n",
    "\n",
    "# 9. Multiplication resulting in pure imaginary\n",
    "c1 = 1+1j\n",
    "c2 = 1+1j\n",
    "result = c1 * c2 # (1+1j)^2 = 1 + 2j + j^2 = 1 + 2j - 1 = 2j\n",
    "print(f\"9. {c1} * {c2} = {result}\") # Output: 2j\n",
    "\n",
    "# 10. Multiplying by 1+0j (identity)\n",
    "identity = complex(1, 0)\n",
    "result = growth_projection * identity\n",
    "print(f\"10. {growth_projection} * {identity} = {result}\") # Output: (10+3j)\n",
    "```\n",
    "*(Examples for `+`, `-`, `/` follow complex arithmetic rules)*\n",
    "\n",

    "### d) `complex.conjugate()`\n",
    "**Explanation:** Method called on a complex number object. Returns the complex conjugate, which has the same real part but an imaginary part with the opposite sign. If `z = a + bj`, its conjugate is `a - bj`.\n",
    "**Examples:**\n",
    "```python\n",
    "growth_projection = 10+3j\n",
    "# 1. Conjugate of growth_projection\n",
    "conj = growth_projection.conjugate()\n",
    "print(f\"1. Conjugate of {growth_projection}: {conj}\") # Output: (10-3j)\n",
    "\n",
    "# 2. Conjugate of a pure imaginary number\n",
    "imag_num = 5j\n",
    "conj = imag_num.conjugate()\n",
    "print(f\"2. Conjugate of {imag_num}: {conj}\") # Output: (-0-5j) or -5j\n",
    "\n",
    "# 3. Conjugate of a real number (no change)\n",
    "real_num = complex(total_students, 0) # 1500+0j\n",
    "conj = real_num.conjugate()\n",
    "print(f\"3. Conjugate of {real_num}: {conj}\") # Output: (1500+0j)\n",
    "\n",
    "# 4. Conjugate of a number with negative imaginary part\n",
    "neg_imag_complex = 4-7j\n",
    "conj = neg_imag_complex.conjugate()\n",
    "print(f\"4. Conjugate of {neg_imag_complex}: {conj}\") # Output: (4+7j)\n",
    "\n",
    "# 5. Conjugate of zero\n",
    "zero_complex = 0j\n",
    "conj = zero_complex.conjugate()\n",
    "print(f\"5. Conjugate of {zero_complex}: {conj}\") # Output: 0j\n",
    "\n",
    "# 6. Conjugate of a conjugate (returns original)\n",
    "conj1 = growth_projection.conjugate() # 10-3j\n",
    "conj2 = conj1.conjugate() # 10+3j\n",
    "print(f\"6. Conjugate of conjugate of {growth_projection}: {conj2}\") # Output: (10+3j)\n",
    "\n",
    "# 7. Conjugate with float components\n",
    "float_complex = complex(4.8, -1.5)\n",
    "conj = float_complex.conjugate()\n",
    "print(f\"7. Conjugate of {float_complex}: {conj}\") # Output: (4.8+1.5j)\n",
    "\n",
    "# 8. Conjugate of a sum: conj(c1+c2) == conj(c1) + conj(c2)\n",
    "c1 = 1+2j\n",
    "c2 = 3-4j\n",
    "sum_conj = (c1 + c2).conjugate() # (4-2j).conjugate() -> 4+2j\n",
    "ind_conj_sum = c1.conjugate() + c2.conjugate() # (1-2j) + (3+4j) -> 4+2j\n",
    "print(f\"8. Conjugate of sum: {sum_conj}, Sum of conjugates: {ind_conj_sum}\") # Output: (4+2j), (4+2j)\n",
    "\n",
    "# 9. Conjugate of a product: conj(c1*c2) == conj(c1) * conj(c2)\n",
    "prod_conj = (c1 * c2).conjugate() # ((1+2j)*(3-4j)).conjugate() = (11+2j).conjugate() = 11-2j\n",
    "ind_conj_prod = c1.conjugate() * c2.conjugate() # (1-2j)*(3+4j) = 11-2j\n",
    "print(f\"9. Conjugate of product: {prod_conj}, Product of conjugates: {ind_conj_prod}\") # Output: (11-2j), (11-2j)\n",
    "\n",
    "# 10. Conjugate of a negative real part number\n",
    "neg_real_complex = -5+2j\n",
    "conj = neg_real_complex.conjugate()\n",
    "print(f\"10. Conjugate of {neg_real_complex}: {conj}\") # Output: (-5-2j)\n",
    "```\n",
    "\n",
    "### e) `abs()` with Complex Numbers\n",
    "**Explanation:** When applied to a complex number `z = a + bj`, `abs(z)` calculates its magnitude (or modulus), which is the distance from the origin (0,0) to the point (a,b) in the complex plane. It's calculated as `sqrt(a^2 + b^2)` and the result is a `float`.\n",
    "**Examples:**\n",
    "```python\n",
    "growth_projection = 10+3j # a=10, b=3\n",
    "# 1. Magnitude of growth_projection\n",
    "magnitude = abs(growth_projection) # sqrt(10^2 + 3^2) = sqrt(100 + 9) = sqrt(109) approx 10.44\n",
    "print(f\"1. Magnitude (abs) of {growth_projection}: {magnitude:.4f}\") # Output: 10.4403\n",
    "\n",
    "# 2. Magnitude of a pure imaginary number\n",
    "imag_num = 5j # a=0, b=5\n",
    "magnitude = abs(imag_num) # sqrt(0^2 + 5^2) = sqrt(25) = 5.0\n",
    "print(f\"2. Magnitude of {imag_num}: {magnitude}\") # Output: 5.0\n",
    "\n",
    "# 3. Magnitude of a real number (absolute value)\n",
    "real_num = complex(-1500, 0) # a=-1500, b=0\n",
    "magnitude = abs(real_num) # sqrt((-1500)^2 + 0^2) = sqrt(1500^2) = 1500.0\n",
    "print(f\"3. Magnitude of {real_num}: {magnitude}\") # Output: 1500.0\n",
    "\n",
    "# 4. Magnitude of 3+4j (common Pythagorean triple)\n",
    "c = 3+4j # a=3, b=4\n",
    "magnitude = abs(c) # sqrt(3^2 + 4^2) = sqrt(9 + 16) = sqrt(25) = 5.0\n",
    "print(f\"4. Magnitude of {c}: {magnitude}\") # Output: 5.0\n",
    "\n",
    "# 5. Magnitude of zero\n",
    "zero_complex = 0j\n",
    "magnitude = abs(zero_complex) # sqrt(0^2 + 0^2) = 0.0\n",
    "print(f\"5. Magnitude of {zero_complex}: {magnitude}\") # Output: 0.0\n",
    "\n",
    "# 6. Magnitude of a complex number and its conjugate are equal\n",
    "conj = growth_projection.conjugate() # 10-3j\n",
    "mag_conj = abs(conj) # sqrt(10^2 + (-3)^2) = sqrt(100 + 9) = sqrt(109)\n",
    "print(f\"6. Magnitude of conjugate {conj}: {mag_conj:.4f}\") # Output: 10.4403\n",
    "\n",
    "# 7. Magnitude involving floats\n",
    "float_complex = complex(1.5, -2.0) # a=1.5, b=-2.0\n",
    "magnitude = abs(float_complex) # sqrt(1.5^2 + (-2.0)^2) = sqrt(2.25 + 4) = sqrt(6.25) = 2.5\n",
    "print(f\"7. Magnitude of {float_complex}: {magnitude}\") # Output: 2.5\n",
    "\n",
    "# 8. abs(c1*c2) == abs(c1) * abs(c2)\n",
    "c1 = 1+1j # abs(c1) = sqrt(1^2+1^2) = sqrt(2)\n",
    "c2 = 2+3j # abs(c2) = sqrt(2^2+3^2) = sqrt(13)\n",
    "prod = c1 * c2 # (1+1j)(2+3j) = (2-3) + (3+2)j = -1+5j\n",
    "abs_prod = abs(prod) # sqrt((-1)^2 + 5^2) = sqrt(1+25) = sqrt(26)\n",
    "abs1_abs2 = abs(c1) * abs(c2) # sqrt(2) * sqrt(13) = sqrt(26)\n",
    "print(f\"8. |product|: {abs_prod:.4f}, |c1|*|c2|: {abs1_abs2:.4f}\") # Output: 5.0990, 5.0990\n",
    "\n",
    "# 9. Magnitude of a complex number with only negative parts\n",
    "neg_complex = -2-7j\n",
    "magnitude = abs(neg_complex) # sqrt((-2)^2 + (-7)^2) = sqrt(4 + 49) = sqrt(53)\n",
    "print(f\"9. Magnitude of {neg_complex}: {magnitude:.4f}\") # Output: 7.2801\n",
    "\n",
    "# 10. Magnitude is always non-negative\n",
    "print(f\"10. Is magnitude of {growth_projection} >= 0? {abs(growth_projection) >= 0}\") # Output: True\n",
    "```\n",
    "\n",
    "### f) `complex()`\n",
    "**Explanation:** The type constructor. Can be used to create complex numbers from real and imaginary parts (which can be int or float), or by parsing a string representation.\n",
    "**Examples:**\n",
    "```python\n",
    "# 1. From two integers (like the initial growth_projection)\n",
    "c = complex(10, 3)\n",
    "print(f\"1. complex(10, 3) = {c}\") # Output: (10+3j)\n",
    "\n",
    "# 2. From two floats\n",
    "c = complex(average_rating, -2.5) # complex(4.8, -2.5)\n",
    "print(f\"2. complex({average_rating}, -2.5) = {c}\") # Output: (4.8-2.5j)\n",
    "\n",
    "# 3. From an integer (imaginary part defaults to 0)\n",
    "c = complex(total_students) # complex(1500)\n",
    "print(f\"3. complex({total_students}) = {c}\") # Output: (1500+0j)\n",
    "\n",
    "# 4. From a float (imaginary part defaults to 0)\n",
    "c = complex(average_rating) # complex(4.8)\n",
    "print(f\"4. complex({average_rating}) = {c}\") # Output: (4.8+0j)\n",
    "\n",
    "# 5. From a string representation (no spaces around +/-)\n",
    "c = complex(\"10+3j\")\n",
    "print(f\"5. complex('10+3j') = {c}\") # Output: (10+3j)\n",
    "c_neg = complex(\"5-2j\")\n",
    "print(f\"   complex('5-2j') = {c_neg}\") # Output: (5-2j)\n",
    "c_imag = complex(\"-7j\")\n",
    "print(f\"   complex('-7j') = {c_imag}\") # Output: -7j\n",
    "\n",
    "# 6. From a string with floats\n",
    "c = complex(\"4.8-2.5j\")\n",
    "print(f\"6. complex('4.8-2.5j') = {c}\") # Output: (4.8-2.5j)\n",
    "\n",
    "# 7. Using keyword arguments\n",
    "c = complex(real=10, imag=3)\n",
    "print(f\"7. complex(real=10, imag=3) = {c}\") # Output: (10+3j)\n",
    "\n",
    "# 8. Creating zero complex number\n",
    "c = complex(0, 0)\n",

    "### c) Indexing (`[]`) and Slicing (`[:]`)\n",
    "**Explanation:**\n",
    "*   **Indexing:** Accessing a single character at a specific position (index). Python uses 0-based indexing (first character is at index 0). Negative indices count from the end (-1 is the last character).\n",
    "*   **Slicing:** Extracting a portion (substring) of the string using `[start:stop:step]`.\n",
    "    *   `start`: The index where the slice begins (inclusive). Defaults to 0.\n",
    "    *   `stop`: The index where the slice ends (exclusive). Defaults to the end of the string.\n",
    "    *   `step`: The increment between indices. Defaults to 1.\n",
    "**Examples:**\n",
    "```python\n",
    "institute = \"Edukron\"\n",
    "tech_stack = \"Azure DevOps and Python\"\n",
    "\n",

    "first_char = institute[0]\n",
    "print(f\"1. First character of '{institute}': {first_char}\") # Output: E\n",
    "\n",
    "# 2. Get the last character of institute\n",
    "last_char = institute[-1]\n",
    "print(f\"2. Last character of '{institute}': {last_char}\") # Output: n\n",
    "\n",
    "# 3. Get the character at index 3\n",
    "char_at_3 = institute[3]\n",
    "print(f\"3. Character at index 3 of '{institute}': {char_at_3}\") # Output: k\n",
    "\n",
    "# --- Slicing ---\n",
    "# 4. Get the first 5 characters of tech_stack (\"Azure\")\n",

    "\n",
    "# 5. Get the substring \"DevOps\" (index 6 to 12, exclusive)\n",

    "print(f\"5. Extracting 'DevOps': '{devops_part}'\") # Output: 'DevOps'\n",
    "\n",
    "# 6. Get the last 6 characters (\"Python\")\n",
    "last_6 = tech_stack[-6:]\n",
    "print(f\"6. Last 6 chars of tech_stack: '{last_6}'\") # Output: 'Python'\n",
    "\n",
    "# 7. Get everything *except* the first and last character\n",
    "middle = institute[1:-1]\n",
    "print(f\"7. Middle part of '{institute}': '{middle}'\") # Output: 'dukro'\n",
    "\n",
    "# 8. Get every second character of institute\n",
    "every_second = institute[::2]\n",
    "print(f\"8. Every second char of '{institute}': '{every_second}'\") # Output: 'Eurn'\n",
    "\n",
    "# 9. Reverse the institute name using slicing\n",

    "print(f\"9. Reversed '{institute}': '{reversed_name}'\") # Output: 'norkudE'\n",

    "**Explanation:**\n",

    "*   `lstrip([chars])`: Removes leading characters only.\n",
    "*   `rstrip([chars])`: Removes trailing characters only.\n",
    "Useful for cleaning up user input or data read from files.\n",
    "**Examples:**\n",
    "```python\n",
    "# 1. Removing leading/trailing whitespace\n",

    "# 2. Removing only leading whitespace\n",
    "leading_space = \"   Edukron\"\n",
    "left_stripped = leading_space.lstrip()\n",

    "\n",
    "# 3. Removing only trailing whitespace\n",
    "trailing_space = \"Python   \"\n",

    "course_code = \"===DS101===\"\n",

    "# 2. Replacing spaces with underscores\n",

    "### g) `str.split(sep=None, maxsplit=-1)`\n",
    "**Explanation:** Returns a *list* of substrings (words) by splitting the string at occurrences of the separator `sep`.\n",
    "*   If `sep` is omitted or `None`, splits by any whitespace character and discards empty strings.\n",
    "*   If `sep` is specified, splits exactly at that separator.\n",
    "*   `maxsplit`: If specified, performs at most `maxsplit` splits (resulting in `maxsplit + 1` elements in the list). The remainder of the string is the last element.\n",
    "**Examples:**\n",
    "```python\n",
    "course_slogan = \"Learn Data Engineering with Real-Time Projects\"\n",
    "tech_stack = \"Azure DevOps,Python,SQL,Git\"\n",
    "institute = \"Edukron\"\n",
    "\n",
    "# 1. Splitting slogan by whitespace (default)\n",
    "words = course_slogan.split()\n",
    "print(f\"1. Slogan words: {words}\")\n",
    "# Output: ['Learn', 'Data', 'Engineering', 'with', 'Real-Time', 'Projects']\n",
    "\n",
    "# 2. Splitting tech stack by comma\n",
    "skills = tech_stack.split(',')\n",
    "print(f\"2. Tech skills: {skills}\")\n",
    "# Output: ['Azure DevOps', 'Python', 'SQL', 'Git']\n",
    "\n",
    "# 3. Splitting institute name by character 'k'\n",
    "parts = institute.split('k')\n",
    "print(f\"3. Splitting '{institute}' by 'k': {parts}\") # Output: ['Edu', 'ron']\n",
    "\n",
    "# 4. Splitting slogan with maxsplit=2\n",
    "split_limit = course_slogan.split(maxsplit=2)\n",
    "print(f\"4. Splitting slogan (maxsplit=2): {split_limit}\")\n",
    "# Output: ['Learn', 'Data', 'Engineering with Real-Time Projects']\n",
    "\n",
    "# 5. Splitting tech stack by comma with maxsplit=1\n",
    "split_tech_limit = tech_stack.split(',', maxsplit=1)\n",
    "print(f\"5. Splitting tech stack (maxsplit=1): {split_tech_limit}\")\n",
    "# Output: ['Azure DevOps', 'Python,SQL,Git']\n",
    "\n",
    "# 6. Splitting a string with multiple spaces (default handles it)\n",
    "multi_space = \"Azure   DevOps  Python\"\n",
    "words = multi_space.split()\n",
    "print(f\"6. Splitting multi-space string: {words}\")\n",
    "# Output: ['Azure', 'DevOps', 'Python']\n",
    "\n",
    "# 7. Splitting by a separator that is not present\n",
    "result = course_slogan.split(\"---\")\n",
    "print(f\"7. Splitting by non-existent separator: {result}\")\n",
    "# Output: ['Learn Data Engineering with Real-Time Projects'] (list with 1 element)\n",
    "\n",
    "# 8. Splitting an empty string\n",
    "result = \"\".split()\n",
    "print(f\"8. Splitting empty string (default sep): {result}\") # Output: []\n",
    "result_sep = \"\".split(',')\n",
    "print(f\"   Splitting empty string (with sep): {result_sep}\") # Output: ['']\n",

    "# 9. Splitting a string that starts/ends with separator (using sep)\n",
    "path = \"/data/science/\"\n",

    "\n",
    "### h) `str.join(iterable)`\n",
    "**Explanation:** The opposite of `split`. Takes an iterable (like a list or tuple) of strings and concatenates its elements, using the string *on which the method is called* as the separator between elements.\n",
    "**Examples:**\n",
    "```python\n",
    "course_list = [\"Data Science\", \"Data Engineering\", \"Azure DevOps\"]\n",
    "tech_skills = (\"Python\", \"SQL\", \"Git\", \"CI/CD\")\n",
    "words = ['Learn', 'Data', 'Engineering']\n",
    "\n",
    "# 1. Joining course list with a comma and space\n",
    "joined_courses = \", \".join(course_list)\n",
    "print(f\"1. Joined courses: '{joined_courses}'\")\n",
    "# Output: 'Data Science, Data Engineering, Azure DevOps'\n",
    "\n",
    "# 2. Joining tech skills with a hyphen\n",
    "joined_skills = \"-\".join(tech_skills)\n",
    "print(f\"2. Joined skills: '{joined_skills}'\")\n",
    "# Output: 'Python-SQL-Git-CI/CD'\n",
    "\n",
    "# 3. Joining words back into a sentence with spaces\n",
    "sentence = \" \".join(words)\n",
    "print(f\"3. Rejoined sentence: '{sentence}'\") # Output: 'Learn Data Engineering'\n",
    "\n",
    "# 4. Joining with an empty string (no separator)\n",
    "institute = \"Edukron\"\n",
    "chars = list(institute) # ['E', 'd', 'u', 'k', 'r', 'o', 'n']\n",
    "joined_chars = \"\".join(chars)\n",
    "print(f\"4. Joining chars with empty string: '{joined_chars}'\") # Output: 'Edukron'\n",
    "\n",
    "# 5. Joining using a multi-character separator\n",
    "separator = \" | \"\n",
    "joined_fancy = separator.join(course_list)\n",
    "print(f\"5. Joining with ' | ': '{joined_fancy}'\")\n",
    "# Output: 'Data Science | Data Engineering | Azure DevOps'\n",
    "\n",
    "# 6. Joining an iterable with only one element\n",
    "single_item_list = [\"Azure DevOps\"]\n",
    "result = \", \".join(single_item_list)\n",
    "print(f\"6. Joining single item list: '{result}'\") # Output: 'Azure DevOps'\n",
    "\n",
    "# 7. Joining an empty iterable\n",
    "empty_list = []\n",
    "result = \", \".join(empty_list)\n",
    "print(f\"7. Joining empty list: '{result}'\") # Output: ''\n",
    "\n",

    "items = [\"Course\", \"Batch101\", \"Module5\"]\n",

    "# 9. Joining using newline as separator\n",

    "# Edukron Status Report\n",
    "# Students: 1500\n",

    "# 10. Joining requires all elements to be strings (TypeError otherwise)\n",
    "mixed_list = [\"Data Science\", 101]\n",

    "    error_join = \", \".join(mixed_list)\n",
    "except TypeError as e:\n",

    "# Output: Error joining list with non-string: sequence item 1: expected str instance, int found\n",

    "\n",
    "### i) `str.startswith(prefix[, start[, end]])` and `str.endswith(suffix[, start[, end]])`\n",
    "**Explanation:**\n",
    "*   `startswith()`: Returns `True` if the string starts with the specified `prefix`, `False` otherwise.\n",
    "*   `endswith()`: Returns `True` if the string ends with the specified `suffix`, `False` otherwise.\n",
    "*   Optional `start` and `end` arguments restrict the check to a slice of the string.\n",
    "*   `prefix`/`suffix` can also be a tuple of strings to check against.\n",
    "**Examples:**\n",
    "```python\n",
    "institute = \"Edukron\"\n",
    "course_slogan = \"Learn Data Engineering with Real-Time Projects\"\n",
    "filename = \"edukron_report.pdf\"\n",
    "\n",
    "# 1. Check if institute starts with \"Edu\"\n",
    "starts_edu = institute.startswith(\"Edu\")\n",
    "print(f\"1. Does '{institute}' start with 'Edu'? {starts_edu}\") # Output: True\n",
    "\n",
    "# 2. Check if institute starts with \"Data\"\n",
    "starts_data = institute.startswith(\"Data\")\n",
    "print(f\"2. Does '{institute}' start with 'Data'? {starts_data}\") # Output: False\n",
    "\n",
    "# 3. Check if slogan ends with \"Projects\"\n",
    "ends_projects = course_slogan.endswith(\"Projects\")\n",
    "print(f\"3. Does slogan end with 'Projects'? {ends_projects}\") # Output: True\n",
    "\n",
    "# 4. Check if filename ends with \".pdf\"\n",
    "ends_pdf = filename.endswith(\".pdf\")\n",
    "print(f\"4. Does '{filename}' end with '.pdf'? {ends_pdf}\") # Output: True\n",
    "\n",
    "# 5. Check if filename ends with \".csv\" or \".pdf\"\n",
    "ends_data_format = filename.endswith((\".csv\", \".pdf\"))\n",
    "print(f\"5. Does '{filename}' end with '.csv' or '.pdf'? {ends_data_format}\") # Output: True\n",
    "\n",
    "# 6. Check if slogan starts with \"Learn\" or \"Explore\"\n",
    "starts_learn_explore = course_slogan.startswith((\"Learn\", \"Explore\"))\n",
    "print(f\"6. Does slogan start with 'Learn' or 'Explore'? {starts_learn_explore}\") # Output: True\n",
    "\n",
    "# 7. Check a slice: Does \"Engineering\" start at index 11 in slogan?\n",
    "slice_check_start = course_slogan.startswith(\"Engineering\", 11)\n",
    "print(f\"7. Does slogan start with 'Engineering' at index 11? {slice_check_start}\") # Output: True\n",
    "\n",
    "# 8. Check a slice: Does \"Real-Time\" end at index 36 in slogan?\n",
    "# Slice [start:end] includes start, excludes end. So we check up to index 36.\n",
    "slice_check_end = course_slogan.endswith(\"Real-Time\", 0, 36)\n",
    "print(f\"8. Does slogan[0:36] end with 'Real-Time'? {slice_check_end}\") # Output: True\n",
    "\n",
    "# 9. Case-sensitive check (starts with 'learn')\n",
    "starts_learn_lower = course_slogan.startswith(\"learn\")\n",
    "print(f\"9. Does slogan start with 'learn' (lowercase)? {starts_learn_lower}\") # Output: False\n",
    "\n",
    "# 10. Check endswith on an empty string\n",
    "ends_anything = \"\".endswith(\"anything\")\n",
    "print(f\"10. Does '' end with 'anything'? {ends_anything}\") # Output: False (but True if suffix is also \"\")\n",
    "```\n",
    "\n",
    "### j) `str.find(sub[, start[, end]])` and `str.index(sub[, start[, end]])`\n",
    "**Explanation:**\n",
    "*   `find()`: Searches for the first occurrence of substring `sub` within the string (or the specified slice). Returns the starting index if found, and **-1** if not found.\n",
    "*   `index()`: Same as `find()`, but raises a **ValueError** if the substring is not found.\n",
    "Use `find()` if you expect the substring might be missing and want to handle it gracefully (check for -1). Use `index()` if you expect the substring to be present and want an error if it's not.\n",
    "**Examples:**\n",
    "```python\n",
    "course_slogan = \"Learn Data Engineering with Real-Time Projects\"\n",
    "tech_stack = \"Azure DevOps and Python\"\n",
    "\n",
    "# 1. Find the index of \"Data\" in the slogan\n",
    "data_pos = course_slogan.find(\"Data\")\n",
    "print(f\"1. Index of 'Data' using find(): {data_pos}\") # Output: 6\n",
    "\n",
    "# 2. Find the index of \"Python\" in tech_stack\n",
    "python_pos = tech_stack.find(\"Python\")\n",
    "print(f\"2. Index of 'Python' using find(): {python_pos}\") # Output: 18\n",
    "\n",
    "# 3. Find a substring that doesn't exist\n",
    "java_pos = course_slogan.find(\"Java\")\n",
    "print(f\"3. Index of 'Java' using find(): {java_pos}\") # Output: -1\n",
    "\n",
    "# 4. Use index() to find \"Azure\" (will succeed)\n",
    "azure_pos = tech_stack.index(\"Azure\")\n",
    "print(f\"4. Index of 'Azure' using index(): {azure_pos}\") # Output: 0\n",
    "\n",
    "# 5. Use index() to find \"Java\" (will fail)\n",
    "try:\n",
    "    java_pos_index = course_slogan.index(\"Java\")\n",
    "except ValueError as e:\n",
    "    print(f\"5. Error using index() to find 'Java': {e}\")\n",
    "# Output: Error using index() to find 'Java': substring not found\n",
    "\n",
    "# 6. Find \"Engineering\" starting search from index 10\n",
    "eng_pos_slice = course_slogan.find(\"Engineering\", 10)\n",
    "print(f\"6. Index of 'Engineering' starting at 10: {eng_pos_slice}\") # Output: 11\n",
    "\n",
    "# 7. Find \"e\" starting search from index 2\n",
    "e_pos_slice = course_slogan.find(\"e\", 2) # First 'e' is at index 1, next is at 12\n",
    "print(f\"7. Index of 'e' starting at 2: {e_pos_slice}\") # Output: 12\n",
    "\n",
    "# 8. Find the last occurrence (using rfind() or rindex())\n",
    "last_e_pos = course_slogan.rfind(\"e\") # rfind searches from the right\n",
    "print(f\"8. Index of last 'e' using rfind(): {last_e_pos}\") # Output: 43\n",
    "\n",
    "# 9. Find using an end index: find 'Data' before index 10\n",
    "data_pos_end = course_slogan.find(\"Data\", 0, 10)\n",
    "print(f\"9. Index of 'Data' before index 10: {data_pos_end}\") # Output: 6\n",
    "\n",
    "# 10. Find in an empty string\n",
    "empty_find = \"\".find(\"a\")\n",
    "print(f\"10. Find 'a' in empty string: {empty_find}\") # Output: -1\n",
    "```\n",

    "**Explanation:** Removes and *returns* the item at the given `index`. If `index` is omitted, it removes and returns the *last* item. Raises an `IndexError` if the list is empty or the index is out of range. Modifies the list in-place.\n",
    "**Examples:**\n",
    "```python\n",

    "last_course = course_list.pop()\n",

    "# Output: Popped last course: 'Data Engineering', List is now: ['Python Basics', 'Data Science']\n",
    "\n",
    "# 2. Pop the item at index 0\n",
    "first_course = course_list.pop(0)\n",
    "print(f\"2. Popped first course: '{first_course}', List is now: {course_list}\")\n",
    "# Output: Popped first course: 'Python Basics', List is now: ['Data Science']\n",
    "\n",
    "# 3. Pop the last task\n",
    "last_task = tasks.pop()\n",
    "print(f\"3. Popped last task: '{last_task}', Tasks left: {tasks}\")\n",
    "# Output: Popped last task: 'Deploy', Tasks left: ['Review Code', 'Update Docs']\n",
    "\n",
    "# 4. Pop the item at index 1\n",
    "middle_task = tasks.pop(1)\n",
    "print(f\"4. Popped task at index 1: '{middle_task}', Tasks left: {tasks}\")\n",
    "# Output: Popped task at index 1: 'Update Docs', Tasks left: ['Review Code']\n",
    "\n",
    "# 5. Pop the only remaining item\n",
    "only_task = tasks.pop()\n",
    "print(f\"5. Popped only task: '{only_task}', Tasks left: {tasks}\")\n",
    "# Output: Popped only task: 'Review Code', Tasks left: []\n",
    "\n",
    "# 6. Attempt to pop from an empty list (causes IndexError)\n",
    "try:\n",

    "# Output: Error popping from empty list: pop from empty list\n",
    "\n",
    "# 7. Pop using a negative index (-1 is last, -2 is second last)\n",

    "# Output: Popped first element again: 10, List is now: [20]\n",
    "\n",

    "#### g) `list.sort(key=None, reverse=False)`\n",
    "**Explanation:** Sorts the items of the list *in-place*.\n",

    "*   `reverse=True`: Sorts in descending order.\n",
    "Items must be comparable (e.g., all numbers or all strings).\n",
    "**Examples:**\n",
    "```python\n",

    "\n",
    "# 2. Sort student counts numerically (ascending)\n",
    "student_counts.sort()\n",
    "print(f\"2. Sorted counts: {student_counts}\") # Output: [350, 380, 420, 450, 500]\n",
    "\n",
    "# 3. Sort student counts in descending order\n",

    "\n",
    "# 4. Sort mixed case courses case-insensitively\n",
    "mixed_case_courses.sort(key=str.lower)\n",

    "# Output: ['azure DevOps', 'Data science', 'Python basics']\n",
    "\n",
    "# 5. Sort list of tuples based on the second element (student count)\n",
    "course_enrollments = [(\"Data Science\", 380), (\"Python Basics\", 500), (\"Azure DevOps\", 420)]\n",

    "print(f\"5. Courses sorted by enrollment: {course_enrollments}\")\n",
    "# Output: [('Data Science', 380), ('Azure DevOps', 420), ('Python Basics', 500)]\n",
    "\n",
    "# 6. Sort descending by enrollment\n",
    "course_enrollments.sort(key=lambda item: item[1], reverse=True)\n",
    "print(f\"6. Courses sorted descending by enrollment: {course_enrollments}\")\n",
    "# Output: [('Python Basics', 500), ('Azure DevOps', 420), ('Data Science', 380)]\n",
    "\n",
    "# 7. Sort list based on length of strings\n",
    "courses = [\"SQL\", \"Python\", \"Azure DevOps\", \"Git\"]\n",
    "courses.sort(key=len)\n",
    "print(f\"7. Courses sorted by length: {courses}\")\n",
    "# Output: ['SQL', 'Git', 'Python', 'Azure DevOps']\n",
    "\n",
    "# 8. Sort list of lists based on the first element\n",
    "batches = [[102, 'DE'], [101, 'DS'], [103, 'ADO']]\n",
    "batches.sort() # Default sorts by first element, then second if first is equal\n",
    "print(f\"8. Batches sorted by ID: {batches}\")\n",
    "# Output: [[101, 'DS'], [102, 'DE'], [103, 'ADO']]\n",
    "\n",
    "# 9. Sort a list that is already sorted\n",
    "already_sorted = [10, 20, 30]\n",
    "already_sorted.sort()\n",
    "print(f\"9. Sorting an already sorted list: {already_sorted}\") # Output: [10, 20, 30]\n",
    "\n",
    "# 10. Attempt to sort list with incompatible types (TypeError)\n",
    "incompatible_list = [\"Data Science\", 420, True]\n",
    "try:\n",
    "    incompatible_list.sort()\n",
    "except TypeError as e:\n",
    "    print(f\"10. Error sorting incompatible types: {e}\")\n",

    "### 2. `tuple`\n",
    "\n",
    "**Explanation:** Tuples are **immutable** (unchangeable), ordered sequences of items. They are defined using parentheses `()`, with items separated by commas. Like lists, they can contain items of different data types. Because they are immutable, they are often used for fixed collections of items and can be used as keys in dictionaries (whereas lists cannot).\n",
    "\n",
    "Your example:\n",
    "*   `tech_skills = (\"Python\", \"SQL\", \"Git\", \"CI/CD\")`\n",
    "\n",
    "**Common Operations & Methods:**\n",
    "\n",
    "Tuples support indexing, slicing, `len()`, and membership testing (`in`, `not in`) just like lists. However, they lack methods that modify the tuple (like `append`, `remove`, `sort`, etc.). They have only two primary methods: `count()` and `index()`.\n",
    "\n",
    "#### a) Creation and Indexing/Slicing\n",
    "**Explanation:** Creating tuples and accessing elements or sub-tuples. Single-element tuples require a trailing comma (e.g., `(item,)`).\n",
    "**Examples:**\n",
    "```python\n",
    "tech_skills = (\"Python\", \"SQL\", \"Git\", \"CI/CD\", \"Python\")\n",
    "course_info = (\"DS101\", \"Data Science\", 15, 4.8) # ID, Name, Weeks, Rating\n",
    "\n",
    "# 1. Create a tuple of skills\n",
    "print(f\"1. Tech skills tuple: {tech_skills}\")\n",
    "\n",
    "# 2. Access the second skill\n",
    "second_skill = tech_skills[1]\n",
    "print(f\"2. Second skill: {second_skill}\") # Output: SQL\n",
    "\n",
    "# 3. Access the last skill\n",
    "last_skill = tech_skills[-1]\n",
    "print(f\"3. Last skill: {last_skill}\") # Output: Python\n",
    "\n",
    "# 4. Get a slice of the first two skills\n",
    "first_two_skills = tech_skills[:2]\n",
    "print(f\"4. First two skills slice: {first_two_skills}\") # Output: ('Python', 'SQL')\n",
    "\n",
    "# 5. Get course name and rating from course_info\n",
    "name_rating = course_info[1], course_info[3] # Creates a new tuple\n",
    "print(f\"5. Course name and rating: {name_rating}\") # Output: ('Data Science', 4.8)\n",
    "\n",
    "# 6. Create a single-element tuple (note the comma)\n",
    "single_tool = (\"Azure\",)\n",
    "print(f\"6. Single element tuple: {single_tool}, Type: {type(single_tool)}\")\n",
    "# Output: ('Azure',), Type: <class 'tuple'>\n",
    "\n",
    "# 7. Create an empty tuple\n",
    "empty_tuple = ()\n",
    "print(f\"7. Empty tuple: {empty_tuple}\")\n",
    "\n",
    "# 8. Tuple packing (creating tuple without parentheses)\n",
    "student_record = \"Alice\", 101, \"DS\" # Automatically becomes a tuple\n",
    "print(f\"8. Tuple packing: {student_record}\") # Output: ('Alice', 101, 'DS')\n",
    "\n",
    "# 9. Tuple unpacking (assigning tuple elements to variables)\n",
    "course_id, course_name, duration, rating = course_info\n",
    "print(f\"9. Unpacked course name: {course_name}, Duration: {duration}\")\n",
    "# Output: Unpacked course name: Data Science, Duration: 15\n",
    "\n",
    "# 10. Accessing elements in a nested tuple\n",
    "nested_tuple = (1, 2, (\"a\", \"b\"), 3)\n",
    "inner_element = nested_tuple[2][1] # Access 'b'\n",
    "print(f\"10. Element in nested tuple: {inner_element}\") # Output: b\n",
    "```\n",
    "\n",
    "#### b) `tuple.count(item)`\n",
    "**Explanation:** Returns the number of times the specified `item` appears in the tuple.\n",
    "**Examples:**\n",
    "```python\n",
    "tech_skills = (\"Python\", \"SQL\", \"Git\", \"CI/CD\", \"Python\", \"SQL\", \"Python\")\n",
    "ratings = (5, 4, 5, 5, 3, 4, 5)\n",
    "\n",
    "# 1. Count occurrences of \"Python\" in tech_skills\n",
    "python_count = tech_skills.count(\"Python\")\n",
    "print(f\"1. Count of 'Python': {python_count}\") # Output: 3\n",
    "\n",
    "# 2. Count occurrences of \"SQL\"\n",
    "sql_count = tech_skills.count(\"SQL\")\n",
    "print(f\"2. Count of 'SQL': {sql_count}\") # Output: 2\n",
    "\n",
    "# 3. Count occurrences of \"Git\"\n",
    "git_count = tech_skills.count(\"Git\")\n",
    "print(f\"3. Count of 'Git': {git_count}\") # Output: 1\n",
    "\n",
    "# 4. Count occurrences of \"Java\" (not present)\n",
    "java_count = tech_skills.count(\"Java\")\n",
    "print(f\"4. Count of 'Java': {java_count}\") # Output: 0\n",
    "\n",
    "# 5. Count occurrences of rating 5\n",
    "rating_5_count = ratings.count(5)\n",
    "print(f\"5. Count of rating 5: {rating_5_count}\") # Output: 4\n",
    "\n",
    "# 6. Count occurrences of rating 3\n",
    "rating_3_count = ratings.count(3)\n",
    "print(f\"6. Count of rating 3: {rating_3_count}\") # Output: 1\n",
    "\n",
    "# 7. Count occurrences of rating 1 (not present)\n",
    "rating_1_count = ratings.count(1)\n",
    "print(f\"7. Count of rating 1: {rating_1_count}\") # Output: 0\n",
    "\n",
    "# 8. Count in a tuple with mixed types\n",
    "mixed_tuple = (1, \"Python\", True, 1, \"Python\", 1)\n",
    "count_1 = mixed_tuple.count(1) # Note: True is treated as 1 here\n",
    "print(f\"8. Count of 1 in mixed tuple: {count_1}\") # Output: 4 (1, True, 1, 1)\n",
    "count_true = mixed_tuple.count(True)\n",
    "print(f\"   Count of True in mixed tuple: {count_true}\") # Output: 4\n",
    "\n",
    "# 9. Count in a tuple with nested tuples (counts the exact tuple)\n",
    "nested = (1, (2, 3), 4, (2, 3))\n",
    "nested_count = nested.count((2, 3))\n",
    "print(f\"9. Count of (2, 3): {nested_count}\") # Output: 2\n",
    "\n",
    "# 10. Count in an empty tuple\n",
    "empty_count = ().count(\"anything\")\n",
    "print(f\"10. Count in empty tuple: {empty_count}\") # Output: 0\n",
    "```\n",
    "\n",
    "#### c) `tuple.index(item[, start[, end]])`\n",
    "**Explanation:** Returns the index of the *first* occurrence of the specified `item`. Raises a `ValueError` if the item is not found. Optional `start` and `end` arguments limit the search to a slice.\n",
    "**Examples:**\n",
    "```python\n",
    "tech_skills = (\"Python\", \"SQL\", \"Git\", \"CI/CD\", \"Python\", \"SQL\")\n",
    "ratings = (5, 4, 5, 5, 3, 4, 5)\n",
    "\n",
    "# 1. Find the index of the first occurrence of \"SQL\"\n",
    "sql_index = tech_skills.index(\"SQL\")\n",
    "print(f\"1. Index of first 'SQL': {sql_index}\") # Output: 1\n",
    "\n",
    "# 2. Find the index of the first occurrence of \"Python\"\n",
    "python_index = tech_skills.index(\"Python\")\n",
    "print(f\"2. Index of first 'Python': {python_index}\") # Output: 0\n",
    "\n",
    "# 3. Find the index of \"Git\"\n",
    "git_index = tech_skills.index(\"Git\")\n",
    "print(f\"3. Index of 'Git': {git_index}\") # Output: 2\n",
    "\n",
    "# 4. Attempt to find the index of \"Java\" (raises ValueError)\n",
    "try:\n",
    "    java_index = tech_skills.index(\"Java\")\n",
    "except ValueError as e:\n",
    "    print(f\"4. Error finding index of 'Java': {e}\")\n",
    "# Output: Error finding index of 'Java': tuple.index(x): x not in tuple\n",
    "\n",
    "# 5. Find the index of the first rating 5\n",
    "rating_5_index = ratings.index(5)\n",
    "print(f\"5. Index of first rating 5: {rating_5_index}\") # Output: 0\n",
    "\n",
    "# 6. Find the index of rating 3\n",
    "rating_3_index = ratings.index(3)\n",
    "print(f\"6. Index of rating 3: {rating_3_index}\") # Output: 4\n",
    "\n",
    "# 7. Find \"Python\" starting the search from index 1\n",
    "python_index_from_1 = tech_skills.index(\"Python\", 1) # Starts search at index 1, finds it at index 4\n",
    "print(f\"7. Index of 'Python' starting from index 1: {python_index_from_1}\") # Output: 4\n",
    "\n",
    "# 8. Find \"SQL\" starting from index 2\n",
    "sql_index_from_2 = tech_skills.index(\"SQL\", 2) # Starts search at index 2, finds it at index 5\n",
    "print(f\"8. Index of 'SQL' starting from index 2: {sql_index_from_2}\") # Output: 5\n",
    "\n",
    "# 9. Find rating 4 between index 2 and 5 (exclusive end)\n",
    "rating_4_slice = ratings.index(4, 2, 5) # Search in (5, 5, 3), finds nothing\n",
    "try:\n",
    "    rating_4_slice = ratings.index(4, 2, 5) # Search in slice (5, 5, 3)\n",
    "except ValueError as e:\n",
    "    print(f\"9. Error finding rating 4 in slice [2:5]: {e}\")\n",
    "# Output: Error finding rating 4 in slice [2:5]: 4 is not in tuple\n",
    "\n",
    "# 10. Find index of a nested tuple\n",
    "nested = (1, (2, 3), 4, (2, 3))\n",
    "nested_index = nested.index((2, 3))\n",
    "print(f\"10. Index of (2, 3): {nested_index}\") # Output: 1\n",
    "```\n",
    "*(Also supports `len()`, `in`, `not in` similar to lists)*\n",
    "\n",
    "---\n",
    "\n",
    "### 3. `range`\n",
    "\n",
    "**Explanation:** Represents an **immutable** sequence of numbers, typically used for looping a specific number of times in `for` loops. It's memory-efficient because it doesn't generate all the numbers at once, only when needed (it's a generator).\n",
    "Created using `range(stop)`, `range(start, stop)`, or `range(start, stop, step)`.\n",
    "*   `start`: The starting number (inclusive, defaults to 0).\n",
    "*   `stop`: The ending number (exclusive).\n",
    "*   `step`: The increment between numbers (defaults to 1).\n",
    "\n",
    "Your example:\n",
    "*   `batch_ids = range(101, 106)` (Represents 101, 102, 103, 104, 105)\n",
    "\n",
    "**Common Operations & Usage:**\n",
    "\n",
    "Ranges primarily support iteration, `len()`, membership testing (`in`, `not in`), indexing, and slicing (though slicing also returns a range object).\n",
    "\n",
    "#### a) Creation and Iteration\n",
    "**Explanation:** Creating range objects and using them in `for` loops.\n",
    "**Examples:**\n",
    "```python\n",
    "# 1. Simple range (0 up to, but not including, 5)\n",
    "print(\"1. Looping with range(5):\")\n",
    "for i in range(5):\n",
    "    print(f\"   - Iteration {i}\")\n",
    "# Output: Iteration 0, 1, 2, 3, 4\n",
    "\n",
    "# 2. Range with start and stop (batch IDs 101 to 105)\n",
    "batch_ids = range(101, 106)\n",
    "print(f\"\\n2. Processing Batch IDs: {batch_ids}\")\n",
    "for batch_id in batch_ids:\n",
    "    print(f\"   - Processing Batch {batch_id}\")\n",
    "# Output: Processing Batch 101, 102, 103, 104, 105\n",
    "\n",
    "# 3. Range with start, stop, and step (even numbers 0 to 8)\n",
    "even_numbers = range(0, 10, 2)\n",
    "print(f\"\\n3. Even numbers range: {even_numbers}\")\n",
    "for num in even_numbers:\n",
    "    print(f\"   - Even number: {num}\")\n",
    "# Output: Even number: 0, 2, 4, 6, 8\n",
    "\n",
    "# 4. Range with negative step (countdown from 5 down to 1)\n",
    "countdown = range(5, 0, -1)\n",
    "print(\"\\n4. Countdown:\")\n",
    "for i in countdown:\n",
    "    print(f\"   - T-minus {i}\")\n",
    "# Output: T-minus 5, 4, 3, 2, 1\n",
    "\n",
    "# 5. Creating a range object directly\n",
    "module_range = range(1, 6) # Represents modules 1 to 5\n",
    "print(f\"\\n5. Module range object: {module_range}\") # Output: range(1, 6)\n",
    "\n",
    "# 6. Iterating through indices of a list\n",
    "course_list = [\"DS\", \"DE\", \"ADO\"]\n",
    "print(\"\\n6. Accessing courses by index:\")\n",
    "for i in range(len(course_list)):\n",
    "    print(f\"   - Index {i}: {course_list[i]}\")\n",
    "# Output: Index 0: DS, Index 1: DE, Index 2: ADO\n",
    "\n",
    "# 7. Range where start equals stop (empty sequence)\n",
    "empty_range = range(5, 5)\n",
    "print(\"\\n7. Iterating over empty range:\")\n",
    "for i in empty_range:\n",
    "    print(f\"   - This won't print: {i}\") # No output from loop\n",
    "print(f\"   (Empty range object: {empty_range})\")\n",
    "\n",
    "# 8. Range where step doesn't allow reaching stop (e.g., positive step when start > stop)\n",
    "invalid_step_range = range(10, 0, 1) # Start=10, Stop=0, Step=1 -> Empty\n",
    "print(\"\\n8. Iterating over invalid step range:\")\n",
    "for i in invalid_step_range:\n",
    "    print(f\"   - This won't print: {i}\") # No output\n",
    "print(f\"   (Invalid step range object: {invalid_step_range})\")\n",
    "\n",
    "# 9. Converting range to list to see all numbers\n",
    "batch_list = list(range(101, 106))\n",
    "print(f\"\\n9. Batch IDs as a list: {batch_list}\") # Output: [101, 102, 103, 104, 105]\n",
    "\n",
    "# 10. Converting range to tuple\n",
    "even_tuple = tuple(range(0, 10, 2))\n",
    "print(f\"10. Even numbers as tuple: {even_tuple}\") # Output: (0, 2, 4, 6, 8)\n",
    "```\n",
    "\n",
    "#### b) `len()`, Membership (`in`, `not in`), Indexing, Slicing\n",
    "**Explanation:** Getting length, checking for number presence, accessing elements by index (without generating all numbers), and slicing range objects.\n",
    "**Examples:**\n",
    "```python\n",
    "batch_ids = range(101, 106) # 101, 102, 103, 104, 105\n",
    "numbers = range(0, 20, 3) # 0, 3, 6, 9, 12, 15, 18\n",
    "\n",
    "# 1. Get the number of items in batch_ids range\n",
    "num_batches = len(batch_ids)\n",
    "print(f\"1. Number of batch IDs: {num_batches}\") # Output: 5\n",
    "\n",
    "# 2. Get the number of items in numbers range\n",
    "count_numbers = len(numbers)\n",
    "print(f\"2. Count of numbers in range(0, 20, 3): {count_numbers}\") # Output: 7\n",
    "\n",
    "# 3. Check if batch ID 103 is in the range\n",
    "is_103_present = 103 in batch_ids\n",
    "print(f\"3. Is batch ID 103 present? {is_103_present}\") # Output: True\n",
    "\n",
    "# 4. Check if batch ID 106 is in the range\n",
    "is_106_present = 106 in batch_ids\n",
    "print(f\"4. Is batch ID 106 present? {is_106_present}\") # Output: False (range stops before 106)\n",
    "\n",
    "# 5. Check if 10 is NOT in the 'numbers' range\n",
    "is_10_not_present = 10 not in numbers\n",
    "print(f\"5. Is 10 not in numbers range? {is_10_not_present}\") # Output: True (only multiples of 3)\n",
    "\n",
    "# 6. Check if 12 IS in the 'numbers' range\n",
    "is_12_present = 12 in numbers\n",
    "print(f\"6. Is 12 in numbers range? {is_12_present}\") # Output: True\n",
    "\n",
    "# 7. Get the first element (index 0) of batch_ids\n",
    "first_batch_id = batch_ids[0]\n",
    "print(f\"7. First batch ID (index 0): {first_batch_id}\") # Output: 101\n",
    "\n",
    "# 8. Get the third element (index 2) of batch_ids\n",
    "third_batch_id = batch_ids[2]\n",
    "print(f\"8. Third batch ID (index 2): {third_batch_id}\") # Output: 103\n",
    "\n",
    "# 9. Get a slice of the batch_ids range (returns another range object)\n",
    "batch_slice = batch_ids[1:4] # Represents indices 1, 2, 3 -> numbers 102, 103, 104\n",
    "print(f\"9. Slice of batch IDs [1:4]: {batch_slice}\") # Output: range(102, 105)\n",
    "print(f\"   Slice as list: {list(batch_slice)}\") # Output: [102, 103, 104]\n",
    "\n",
    "# 10. Get the last element using negative index\n",
    "last_batch_id = batch_ids[-1]\n",
    "print(f\"10. Last batch ID (index -1): {last_batch_id}\") # Output: 105\n",
    "```\n",
    "\n",
    "---\n",
    "\n",
    "##  Set Types\n",
    "\n",
    "Set types represent unordered collections of **unique** items. They are useful for membership testing, removing duplicates, and performing mathematical set operations (like union, intersection, difference).\n",
    "\n",
    "### 1. `set`\n",
    "\n",
    "**Explanation:** Sets are **mutable** (changeable), unordered collections of unique, **hashable** items. Hashable means the items must be immutable (like numbers, strings, tuples, but not lists or other sets). Sets are defined using curly braces `{}` or the `set()` constructor. An empty set *must* be created with `set()`, as `{}` creates an empty dictionary.\n",
    "\n",
    "Your examples:\n",
    "*   `data_analytics_topics = {\"Visualization\", \"Power BI\", \"EDA\", \"Reporting\"}`\n",
    "*   `devops_tools = {\"Docker\", \"Kubernetes\", \"Terraform\", \"GitHub Actions\"}`\n",
    "\n",
    "**Common Operations & Methods:**\n",
    "\n",
    "#### a) Creation and `add(item)`\n",
    "**Explanation:** Creating sets (removes duplicates automatically) and adding single elements.\n",
    "**Examples:**\n",
    "```python\n",
    "# 1. Create a set of analytics topics (duplicates ignored)\n",
    "data_analytics_topics = {\"Visualization\", \"Power BI\", \"EDA\", \"Reporting\", \"Power BI\"}\n",
    "print(f\"1. Analytics topics set: {data_analytics_topics}\")\n",
    "# Output: {'Visualization', 'EDA', 'Reporting', 'Power BI'} (Order may vary)\n",
    "\n",
    "# 2. Create a set using set() from a list (removes duplicates)\n",
    "course_list = [\"Data Science\", \"Data Engineering\", \"Azure DevOps\", \"Data Science\"]\n",
    "unique_courses = set(course_list)\n",
    "print(f\"2. Unique courses from list: {unique_courses}\")\n",
    "# Output: {'Data Engineering', 'Azure DevOps', 'Data Science'} (Order may vary)\n",
    "\n",
    "# 3. Create an empty set\n",
    "required_skills = set()\n",
    "print(f\"3. Empty set: {required_skills}\")\n",
    "\n",
    "# 4. Add a skill to the empty set\n",
    "required_skills.add(\"Python\")\n",
    "print(f\"4. Set after adding 'Python': {required_skills}\") # Output: {'Python'}\n",
    "\n",
    "# 5. Add another skill\n",
    "required_skills.add(\"SQL\")\n",
    "print(f\"5. Set after adding 'SQL': {required_skills}\") # Output: {'Python', 'SQL'} (Order may vary)\n",
    "\n",
    "# 6. Add an existing skill (set remains unchanged)\n",
    "required_skills.add(\"Python\")\n",
    "print(f\"6. Set after adding 'Python' again: {required_skills}\") # Output: {'Python', 'SQL'}\n",
    "\n",
    "# 7. Add a number to a set\n",
    "batch_ids = {101, 102}\n",
    "batch_ids.add(103)\n",
    "print(f\"7. Batch IDs after add: {batch_ids}\") # Output: {101, 102, 103}\n",
    "\n",
    "# 8. Add a tuple (hashable) to a set\n",
    "config_params = {(\"host\", \"localhost\")}\n",
    "config_params.add((\"port\", 8080))\n",
    "print(f\"8. Config params set: {config_params}\")\n",
    "# Output: {('host', 'localhost'), ('port', 8080)}\n",
    "\n",
    "# 9. Create set from a string (unique characters)\n",
    "unique_chars = set(\"Edukron Institute\")\n",
    "print(f\"9. Unique chars in 'Edukron Institute': {unique_chars}\")\n",
    "# Output: {'r', 's', 'o', 't', 'n', 'k', 'I', 'u', 'd', ' ', 'E', 'i'} (Order may vary)\n",
    "\n",
    "# 10. Attempt to add unhashable type (list) -> TypeError\n",
    "try:\n",
    "    required_skills.add([\"Git\", \"CI/CD\"])\n",
    "except TypeError as e:\n",
    "    print(f\"10. Error adding list to set: {e}\")\n",
    "# Output: Error adding list to set: unhashable type: 'list'\n",
    "```\n",
    "\n",
    "#### b) `update(iterable)`\n",
    "**Explanation:** Adds all unique items from an `iterable` (like a list, tuple, set, or string) into the set. Modifies the set in-place. Equivalent to `set1 |= set2 | set3 ...`\n",
    "**Examples:**\n",
    "```python\n",
    "devops_tools = {\"Docker\", \"Kubernetes\"}\n",
    "data_tools = {\"Python\", \"SQL\", \"Pandas\"}\n",
    "new_tools_list = [\"Terraform\", \"GitHub Actions\", \"Docker\"] # Note duplicate \"Docker\"\n",
    "more_skills = (\"Git\", \"SQL\") # Note duplicate \"SQL\"\n",
    "\n",
    "# 1. Update devops_tools with new_tools_list\n",
    "devops_tools.update(new_tools_list)\n",
    "print(f\"1. DevOps tools after update: {devops_tools}\")\n",
    "# Output: {'GitHub Actions', 'Docker', 'Terraform', 'Kubernetes'} (Order may vary)\n",
    "\n",
    "# 2. Update data_tools with more_skills tuple\n",
    "data_tools.update(more_skills)\n",
    "print(f\"2. Data tools after update: {data_tools}\")\n",
    "# Output: {'Pandas', 'Git', 'SQL', 'Python'} (Order may vary)\n",
    "\n",
    "# 3. Update with another set\n",
    "all_tools = data_tools.copy() # Start with data tools\n",
    "all_tools.update(devops_tools)\n",
    "print(f\"3. All tools combined using update: {all_tools}\")\n",
    "# Output: {'Pandas', 'Git', 'SQL', 'Python', 'GitHub Actions', 'Docker', 'Terraform', 'Kubernetes'}\n",
    "\n",
    "# 4. Update with characters from a string\n",
    "chars = {'a', 'b'}\n",
    "chars.update(\"ace\") # Adds 'c', 'e' ('a' already exists)\n",
    "print(f\"4. Chars updated with string 'ace': {chars}\") # Output: {'b', 'c', 'a', 'e'}\n",
    "\n",
    "# 5. Update with an empty iterable (no change)\n",
    "initial_set = {1, 2}\n",
    "initial_set.update([])\n",
    "print(f\"5. Set updated with empty list: {initial_set}\") # Output: {1, 2}\n",
    "\n",
    "# 6. Update an empty set\n",
    "empty_set = set()\n",
    "empty_set.update([101, 102, 101])\n",
    "print(f\"6. Updating an empty set: {empty_set}\") # Output: {101, 102}\n",
    "\n",
    "# 7. Update with items from a range\n",
    "numbers = {1, 5}\n",
    "numbers.update(range(3, 6)) # Adds 3, 4 (5 already exists)\n",
    "print(f\"7. Numbers updated with range(3, 6): {numbers}\") # Output: {1, 3, 4, 5}\n",
    "\n",
    "# 8. Update with multiple iterables at once\n",
    "features = {\"Login\"}\n",
    "features.update([\"Profile\", \"Settings\"], (\"Logout\", \"Profile\"))\n",
    "print(f\"8. Features updated with multiple iterables: {features}\")\n",
    "# Output: {'Logout', 'Settings', 'Profile', 'Login'}\n",
    "\n",
    "# 9. Update using the |= operator (equivalent)\n",
    "set_a = {1, 2}\n",
    "set_b = {2, 3}\n",
    "set_a |= set_b # set_a = set_a.union(set_b)\n",
    "print(f\"9. Set A updated using |= : {set_a}\") # Output: {1, 2, 3}\n",
    "\n",
    "# 10. Attempt to update with non-iterable (TypeError)\n",
    "try:\n",
    "    tools = {\"Git\"}\n",
    "    tools.update(123)\n",
    "except TypeError as e:\n",
    "    print(f\"10. Error updating with non-iterable: {e}\")\n",
    "# Output: Error updating with non-iterable: 'int' object is not iterable\n",
    "```\n",
    "\n",
    "#### c) `remove(item)` vs `discard(item)`\n",
    "**Explanation:** Both remove an item from the set.\n",
    "*   `remove(item)`: Removes `item`. Raises a `KeyError` if `item` is not found.\n",
    "*   `discard(item)`: Removes `item` if present. Does *not* raise an error if `item` is not found.\n",
    "Both modify the set in-place.\n",
    "**Examples:**\n",
    "```python\n",
    "devops_tools = {\"Docker\", \"Kubernetes\", \"Terraform\", \"GitHub Actions\"}\n",
    "\n",
    "# 1. Remove \"Terraform\" using remove()\n",
    "devops_tools.remove(\"Terraform\")\n",
    "print(f\"1. Set after removing 'Terraform': {devops_tools}\")\n",
    "# Output: {'GitHub Actions', 'Kubernetes', 'Docker'} (Order may vary)\n",
    "\n",
    "# 2. Attempt to remove \"Terraform\" again using remove() (KeyError)\n",
    "try:\n",
    "    devops_tools.remove(\"Terraform\")\n",
    "except KeyError as e:\n",
    "    print(f\"2. Error removing 'Terraform' again: {e}\")\n",
    "# Output: Error removing 'Terraform' again: 'Terraform'\n",
    "\n",
    "# 3. Discard \"Kubernetes\" using discard()\n",
    "devops_tools.discard(\"Kubernetes\")\n",
    "print(f\"3. Set after discarding 'Kubernetes': {devops_tools}\")\n",
    "# Output: {'GitHub Actions', 'Docker'}\n",
    "\n",
    "# 4. Attempt to discard \"Kubernetes\" again (no error)\n",
    "devops_tools.discard(\"Kubernetes\")\n",
    "print(f\"4. Set after discarding 'Kubernetes' again: {devops_tools}\") # No change, no error\n",
    "# Output: {'GitHub Actions', 'Docker'}\n",
    "\n",
    "# 5. Discard an item not present (\"Ansible\")\n",
    "devops_tools.discard(\"Ansible\")\n",
    "print(f\"5. Set after discarding 'Ansible' (not present): {devops_tools}\") # No change, no error\n",
    "# Output: {'GitHub Actions', 'Docker'}\n",
    "\n",
    "# 6. Remove \"Docker\" using remove()\n",
    "devops_tools.remove(\"Docker\")\n",
    "print(f\"6. Set after removing 'Docker': {devops_tools}\")\n",
    "# Output: {'GitHub Actions'}\n",
    "\n",
    "# 7. Remove the last item \"GitHub Actions\"\n",
    "devops_tools.remove(\"GitHub Actions\")\n",
    "print(f\"7. Set after removing last item: {devops_tools}\") # Output: set()\n",
    "\n",
    "# 8. Discard from an empty set (no error)\n",
    "devops_tools.discard(\"Anything\")\n",
    "print(f\"8. Discarding from empty set: {devops_tools}\") # Output: set()\n",
    "\n",
    "# 9. Remove from an empty set (KeyError)\n",
    "try:\n",
    "    devops_tools.remove(\"Anything\")\n",
    "except KeyError as e:\n",
    "    print(f\"9. Error removing from empty set: {e}\")\n",
    "# Output: Error removing from empty set: 'Anything'\n",
    "\n",
    "# 10. Conditional removal using discard\n",
    "topics = {\"EDA\", \"Visualization\", \"Power BI\"}\n",
    "topic_to_remove = \"SQL\" # Might not be present\n",
    "topics.discard(topic_to_remove) # Safely removes if present, does nothing otherwise\n",
    "print(f\"10. Topics after safely discarding '{topic_to_remove}': {topics}\")\n",
    "# Output: {'Power BI', 'EDA', 'Visualization'}\n",
    "```\n",
    "\n",
    "#### d) `pop()`\n",
    "**Explanation:** Removes and *returns* an **arbitrary** element from the set. Raises a `KeyError` if the set is empty. Since sets are unordered, you cannot predict which element will be popped. Modifies the set in-place.\n",
    "**Examples:**\n",
    "```python\n",
    "data_analytics_topics = {\"Visualization\", \"Power BI\", \"EDA\", \"Reporting\"}\n",
    "numbers = {101, 102, 103}\n",
    "\n",
    "# 1. Pop an arbitrary topic\n",
    "popped_topic = data_analytics_topics.pop()\n",
    "print(f\"1. Popped topic: '{popped_topic}', Remaining: {data_analytics_topics}\")\n",
    "# Output might be: Popped topic: 'Visualization', Remaining: {'EDA', 'Reporting', 'Power BI'} (Order/Item vary)\n",
    "\n",
    "# 2. Pop another arbitrary topic\n",
    "popped_topic_2 = data_analytics_topics.pop()\n",
    "print(f\"2. Popped another: '{popped_topic_2}', Remaining: {data_analytics_topics}\")\n",
    "# Output might be: Popped another: 'EDA', Remaining: {'Reporting', 'Power BI'}\n",
    "\n",
    "# 3. Pop an arbitrary number\n",
    "popped_number = numbers.pop()\n",
    "print(f\"3. Popped number: {popped_number}, Remaining: {numbers}\") # e.g., 101, {102, 103}\n",
    "\n",
    "# 4. Pop another number\n",
    "popped_number_2 = numbers.pop()\n",
    "print(f\"4. Popped another number: {popped_number_2}, Remaining: {numbers}\") # e.g., 102, {103}\n",
    "\n",
    "# 5. Pop the last remaining number\n",
    "last_number = numbers.pop()\n",
    "print(f\"5. Popped last number: {last_number}, Remaining: {numbers}\") # e.g., 103, set()\n",
    "\n",
    "# 6. Attempt to pop from the now empty set (KeyError)\n",
    "try:\n",
    "    numbers.pop()\n",
    "except KeyError as e:\n",
    "    print(f\"6. Error popping from empty set: {e}\")\n",
    "# Output: Error popping from empty set: 'pop from an empty set'\n",
    "\n",
    "# 7. Pop from a set with one element\n",
    "single_set = {\"Edukron\"}\n",
    "item = single_set.pop()\n",
    "print(f\"7. Popped from single set: '{item}', Remaining: {single_set}\")\n",
    "# Output: Popped from single set: 'Edukron', Remaining: set()\n",
    "\n",
    "# 8. Use pop in a loop to process all items (order not guaranteed)\n",
    "topics_to_process = {\"Git\", \"SQL\", \"Python\"}\n",
    "print(\"8. Processing topics using pop:\")\n",
    "while topics_to_process: # While set is not empty\n",
    "    topic = topics_to_process.pop()\n",
    "    print(f\"   - Processing: {topic}\")\n",
    "# Output: Order will vary, e.g.: Processing: Git, Processing: SQL, Processing: Python\n",
    "\n",
    "# 9. Pop from a set containing tuples\n",
    "coord_set = {(0,0), (1,1), (2,2)}\n",
    "popped_coord = coord_set.pop()\n",
    "print(f\"9. Popped coordinate: {popped_coord}, Remaining: {coord_set}\")\n",
    "\n",
    "# 10. Pop from a set containing mixed types\n",
    "mixed_set = {1, \"hello\", None, True} # True might collide with 1\n",
    "popped_item = mixed_set.pop()\n",
    "print(f\"10. Popped from mixed set: {popped_item}, Type: {type(popped_item)}, Remaining: {mixed_set}\")\n",
    "```\n",
    "\n",
    "#### e) Set Operations (Union, Intersection, Difference, Symmetric Difference)\n",
    "**Explanation:** Mathematical operations on sets.\n",
    "*   **Union (`|` or `union()`):** Returns a new set with all items from both sets.\n",
    "*   **Intersection (`&` or `intersection()`):** Returns a new set with items common to both sets.\n",
    "*   **Difference (`-` or `difference()`):** Returns a new set with items in the first set but *not* in the second.\n",
    "*   **Symmetric Difference (`^` or `symmetric_difference()`):** Returns a new set with items in either set, but *not* in both.\n",
    "In-place versions exist for some (e.g., `update()` is like union, `intersection_update()`, `difference_update()`, `symmetric_difference_update()`).\n",
    "**Examples:**\n",
    "```python\n",
    "data_sci_skills = {\"Python\", \"Pandas\", \"ML\", \"SQL\"}\n",
    "data_eng_skills = {\"Python\", \"SQL\", \"ADF\", \"ADLS\", \"Databricks\"}\n",
    "devops_skills = {\"GitHub\", \"Pipelines\", \"Terraform\", \"Python\"}\n",
    "\n",
    "# 1. Union of DS and DE skills (all unique skills)\n",
    "all_data_skills = data_sci_skills.union(data_eng_skills)\n",
    "# or: all_data_skills = data_sci_skills | data_eng_skills\n",
    "print(f\"1. Union (DS | DE): {all_data_skills}\")\n",
    "# Output: {'ML', 'ADLS', 'Python', 'ADF', 'Pandas', 'Databricks', 'SQL'} (Order varies)\n",
    "\n",
    "# 2. Intersection of DS and DE skills (common skills)\n",
    "common_data_skills = data_sci_skills.intersection(data_eng_skills)\n",
    "# or: common_data_skills = data_sci_skills & data_eng_skills\n",
    "print(f\"2. Intersection (DS & DE): {common_data_skills}\")\n",
    "# Output: {'Python', 'SQL'}\n",
    "\n",
    "# 3. Skills unique to Data Science (DS - DE)\n",
    "ds_only_skills = data_sci_skills.difference(data_eng_skills)\n",
    "# or: ds_only_skills = data_sci_skills - data_eng_skills\n",
    "print(f\"3. Difference (DS - DE): {ds_only_skills}\")\n",
    "# Output: {'Pandas', 'ML'}\n",
    "\n",
    "# 4. Skills unique to Data Engineering (DE - DS)\n",
    "de_only_skills = data_eng_skills.difference(data_sci_skills)\n",
    "# or: de_only_skills = data_eng_skills - data_sci_skills\n",
    "print(f\"4. Difference (DE - DS): {de_only_skills}\")\n",
    "# Output: {'ADF', 'Databricks', 'ADLS'}\n",
    "\n",
    "# 5. Symmetric Difference (skills in DS or DE, but not both)\n",
    "non_common_data_skills = data_sci_skills.symmetric_difference(data_eng_skills)\n",
    "# or: non_common_data_skills = data_sci_skills ^ data_eng_skills\n",
    "print(f\"5. Symmetric Difference (DS ^ DE): {non_common_data_skills}\")\n",
    "# Output: {'ML', 'ADLS', 'ADF', 'Pandas', 'Databricks'}\n",
    "\n",
    "# 6. Intersection of all three skill sets\n",
    "common_all_three = data_sci_skills & data_eng_skills & devops_skills\n",
    "print(f\"6. Intersection (DS & DE & DevOps): {common_all_three}\")\n",
    "# Output: {'Python'}\n",
    "\n",
    "# 7. Skills in DS but not in DevOps (DS - DevOps)\n",
    "ds_not_devops = data_sci_skills - devops_skills\n",
    "print(f\"7. Difference (DS - DevOps): {ds_not_devops}\")\n",
    "# Output: {'Pandas', 'ML', 'SQL'}\n",
    "\n",
    "# 8. Update DS skills in-place with DE skills (DS becomes the union)\n",
    "ds_skills_copy = data_sci_skills.copy()\n",
    "ds_skills_copy.update(data_eng_skills) # Or ds_skills_copy |= data_eng_skills\n",
    "print(f\"8. DS skills updated with DE skills: {ds_skills_copy}\")\n",
    "# Output: {'ML', 'ADLS', 'Python', 'ADF', 'Pandas', 'Databricks', 'SQL'}\n",
    "\n",
    "# 9. Keep only common skills in DS (in-place intersection)\n",
    "ds_skills_copy = data_sci_skills.copy()\n",
    "ds_skills_copy.intersection_update(data_eng_skills) # Or ds_skills_copy &= data_eng_skills\n",
    "print(f\"9. DS skills updated to intersection with DE: {ds_skills_copy}\")\n",
    "# Output: {'SQL', 'Python'}\n",
    "\n",
    "# 10. Remove skills found in DevOps from DS (in-place difference)\n",
    "ds_skills_copy = data_sci_skills.copy()\n",
    "ds_skills_copy.difference_update(devops_skills) # Or ds_skills_copy -= devops_skills\n",
    "print(f\"10. DS skills after removing DevOps skills: {ds_skills_copy}\")\n",
    "# Output: {'ML', 'Pandas', 'SQL'}\n",
    "```\n",
    "*(Also supports `len()`, `in`, `not in`)*\n",
    "\n",
    "---\n",
    "\n",
    "### 2. `frozenset`\n",
    "\n",
    "**Explanation:** Frozensets are **immutable** (unchangeable), unordered collections of unique, hashable items. They are created using the `frozenset()` constructor. Because they are immutable and hashable, they can be used as elements within other sets or as keys in dictionaries.\n",
    "\n",
    "Your example:\n",
    "*   `platforms = frozenset([\"Azure\", \"Python\", \"Databricks\"])`\n",
    "\n",
    "**Common Operations & Usage:**\n",
    "\n",
    "Frozensets support non-modifying set operations (union, intersection, difference, etc. - returning new frozensets or sets), `len()`, and membership testing (`in`, `not in`). They do *not* support methods like `add`, `remove`, `update`, `pop`, etc.\n",
    "\n",
    "#### a) Creation and Immutability\n",
    "**Explanation:** Creating frozensets and demonstrating their immutability.\n",
    "**Examples:**\n",
    "```python\n",
    "# 1. Create a frozenset from a list\n",
    "platforms = frozenset([\"Azure\", \"Python\", \"Databricks\", \"Azure\"]) # Duplicate ignored\n",
    "print(f\"1. Platforms frozenset: {platforms}\")\n",
    "# Output: frozenset({'Python', 'Databricks', 'Azure'}) (Order may vary)\n",
    "\n",
    "# 2. Create a frozenset from a tuple\n",
    "core_skills = frozenset((\"SQL\", \"Python\"))\n",
    "print(f\"2. Core skills frozenset: {core_skills}\")\n",
    "# Output: frozenset({'SQL', 'Python'})\n",
    "\n",
    "# 3. Create from a string (unique chars)\n",
    "fs_chars = frozenset(\"Edukron\")\n",
    "print(f\"3. Frozenset from 'Edukron': {fs_chars}\")\n",
    "# Output: frozenset({'k', 'n', 'd', 'E', 'r', 'o', 'u'})\n",
    "\n",
    "# 4. Create an empty frozenset\n",
    "empty_fs = frozenset()\n",
    "print(f\"4. Empty frozenset: {empty_fs}\") # Output: frozenset()\n",
    "\n",
    "# 5. Attempt to add an element (AttributeError)\n",
    "try:\n",
    "    platforms.add(\"AWS\")\n",
    "except AttributeError as e:\n",
    "    print(f\"5. Error adding to frozenset: {e}\")\n",
    "# Output: Error adding to frozenset: 'frozenset' object has no attribute 'add'\n",
    "\n",
    "# 6. Attempt to remove an element (AttributeError)\n",
    "try:\n",
    "    platforms.remove(\"Azure\")\n",
    "except AttributeError as e:\n",
    "    print(f\"6. Error removing from frozenset: {e}\")\n",
    "# Output: Error removing from frozenset: 'frozenset' object has no attribute 'remove'\n",
    "\n",
    "# 7. Attempt to update (AttributeError)\n",
    "try:\n",
    "    platforms.update([\"AWS\"])\n",
    "except AttributeError as e:\n",
    "    print(f\"7. Error updating frozenset: {e}\")\n",
    "# Output: Error updating frozenset: 'frozenset' object has no attribute 'update'\n",
    "\n",
    "# 8. Create a set containing frozensets (possible because frozensets are hashable)\n",
    "skill_combinations = {core_skills, frozenset([\"ML\", \"Python\"]), frozenset([\"SQL\"])}\n",
    "print(f\"8. Set of frozensets: {skill_combinations}\")\n",
    "# Output: {frozenset({'SQL'}), frozenset({'Python', 'SQL'}), frozenset({'ML', 'Python'})}\n",
    "\n",
    "# 9. Use frozenset as dictionary key\n",
    "course_tools = {\n",
    "    frozenset([\"Python\", \"Pandas\"]): \"Data Analysis basics\",\n",
    "    platforms: \"Cloud Data Engineering platform\"\n",
    "}\n",
    "print(f\"9. Dict with frozenset keys: {course_tools}\")\n",
    "print(f\"   Value for platforms key: {course_tools[platforms]}\")\n",
    "# Output: Cloud Data Engineering platform\n",
    "\n",
    "# 10. Frozenset from range\n",
    "fs_range = frozenset(range(3)) # 0, 1, 2\n",
    "print(f\"10. Frozenset from range(3): {fs_range}\") # Output: frozenset({0, 1, 2})\n",
    "```\n",
    "\n",
    "#### b) Set Operations with Frozensets\n",
    "**Explanation:** Performing set operations; the result type depends on the operand types (frozenset op frozenset -> frozenset, frozenset op set -> set).\n",
    "**Examples:**\n",
    "```python\n",
    "platforms = frozenset([\"Azure\", \"Python\", \"Databricks\"])\n",
    "cloud_tools = frozenset([\"Azure\", \"AWS\", \"GCP\"])\n",
    "scripting = {\"Python\", \"Bash\"} # A regular set\n",
    "\n",
    "# 1. Union of two frozensets (returns frozenset)\n",
    "all_platforms = platforms.union(cloud_tools)\n",
    "# or: all_platforms = platforms | cloud_tools\n",
    "print(f\"1. Union (fs | fs): {all_platforms}, Type: {type(all_platforms)}\")\n",
    "# Output: frozenset({'AWS', 'GCP', 'Azure', 'Python', 'Databricks'}), Type: <class 'frozenset'>\n",
    "\n",
    "# 2. Intersection of two frozensets (returns frozenset)\n",
    "common_platforms = platforms.intersection(cloud_tools)\n",
    "# or: common_platforms = platforms & cloud_tools\n",
    "print(f\"2. Intersection (fs & fs): {common_platforms}, Type: {type(common_platforms)}\")\n",
    "# Output: frozenset({'Azure'}), Type: <class 'frozenset'>\n",
    "\n",
    "# 3. Difference between two frozensets (returns frozenset)\n",
    "platforms_only = platforms.difference(cloud_tools)\n",
    "# or: platforms_only = platforms - cloud_tools\n",
    "print(f\"3. Difference (platforms - cloud_tools): {platforms_only}, Type: {type(platforms_only)}\")\n",
    "# Output: frozenset({'Python', 'Databricks'}), Type: <class 'frozenset'>\n",
    "\n",
    "# 4. Symmetric difference between two frozensets (returns frozenset)\n",
    "non_common_platforms = platforms.symmetric_difference(cloud_tools)\n",
    "# or: non_common_platforms = platforms ^ cloud_tools\n",
    "print(f\"4. Symmetric Difference (fs ^ fs): {non_common_platforms}, Type: {type(non_common_platforms)}\")\n",
    "# Output: frozenset({'AWS', 'GCP', 'Python', 'Databricks'}), Type: <class 'frozenset'>\n",
    "\n",
    "# 5. Union of frozenset and set (returns set)\n",
    "platform_scripting = platforms.union(scripting)\n",
    "# or: platform_scripting = platforms | scripting\n",
    "print(f\"5. Union (fs | set): {platform_scripting}, Type: {type(platform_scripting)}\")\n",
    "# Output: {'Databricks', 'Azure', 'Bash', 'Python'}, Type: <class 'set'>\n",
    "\n",
    "# 6. Intersection of frozenset and set (returns set)\n",
    "common_platform_scripting = platforms.intersection(scripting)\n",
    "# or: common_platform_scripting = platforms & scripting\n",
    "print(f\"6. Intersection (fs & set): {common_platform_scripting}, Type: {type(common_platform_scripting)}\")\n",
    "# Output: {'Python'}, Type: <class 'set'>\n",
    "\n",
    "# 7. Difference (frozenset - set) (returns set)\n",
    "platforms_not_scripting = platforms.difference(scripting)\n",
    "# or: platforms_not_scripting = platforms - scripting\n",
    "print(f\"7. Difference (fs - set): {platforms_not_scripting}, Type: {type(platforms_not_scripting)}\")\n",
    "# Output: {'Azure', 'Databricks'}, Type: <class 'set'>\n",
    "\n",
    "# 8. Symmetric Difference (frozenset ^ set) (returns set)\n",
    "non_common_platform_scripting = platforms.symmetric_difference(scripting)\n",
    "# or: non_common_platform_scripting = platforms ^ scripting\n",
    "print(f\"8. Symmetric Difference (fs ^ set): {non_common_platform_scripting}, Type: {type(non_common_platform_scripting)}\")\n",
    "# Output: {'Databricks', 'Azure', 'Bash'}, Type: <class 'set'>\n",
    "\n",
    "# 9. Checking subset relationship\n",
    "is_subset = frozenset([\"Python\", \"Azure\"]).issubset(platforms)\n",
    "print(f\"9. Is {{'Python', 'Azure'}} subset of platforms? {is_subset}\") # Output: True\n",
    "\n",
    "# 10. Checking superset relationship\n",
    "is_superset = platforms.issuperset({\"Python\"})\n",
    "print(f\"10. Is platforms superset of {{'Python'}}? {is_superset}\") # Output: True\n",
    "```\n",
    "*(Also supports `len()`, `in`, `not in`)*\n",
    "\n",
    "---\n",
    "\n",
    "##  Mapping Type (`dict`)\n",
    "\n",
    "**Explanation:** Dictionaries (`dict`) store collections of **key-value pairs**. Keys must be unique and **hashable** (immutable types like strings, numbers, tuples containing only hashables). Values can be of any type and are accessed using their corresponding key. Dictionaries are **mutable**. As of Python 3.7+, dictionaries remember the insertion order of items.\n",
    "\n",
    "Your example:\n",
    "```python\n",
    "learning_paths = {\n",
    "    \"Data Science\": [\"Python\", \"Pandas\", \"ML\"],\n",
    "    \"Data Engineering\": [\"ADF\", \"ADLS\", \"Databricks\"],\n",
    "    \"Azure DevOps\": [\"GitHub\", \"Pipelines\", \"Terraform\"]\n",
    "}\n",
    "```\n",
    "\n",
    "**Common Operations & Methods:**\n",
    "\n",
    "#### a) Creation, Accessing (`[]`, `get()`), Adding/Updating\n",
    "**Explanation:** Creating dictionaries, retrieving values using keys, and adding new key-value pairs or updating existing ones. `[]` raises `KeyError` if key is missing, `get(key, default=None)` returns `None` (or a specified default) instead.\n",
    "**Examples:**\n",
    "```python\n",
    "learning_paths = {\n",
    "    \"Data Science\": [\"Python\", \"Pandas\", \"ML\"],\n",
    "    \"Data Engineering\": [\"ADF\", \"ADLS\", \"Databricks\"],\n",
    "}\n",
    "student_ratings = {\"Alice\": 4.5, \"Bob\": 4.8}\n",
    "\n",
    "# 1. Create a dictionary of learning paths\n",
    "print(f\"1. Learning paths: {learning_paths}\")\n",
    "\n",
    "# 2. Access the tools for Data Science using []\n",
    "ds_tools = learning_paths[\"Data Science\"]\n",
    "print(f\"2. Data Science tools: {ds_tools}\") # Output: ['Python', 'Pandas', 'ML']\n",
    "\n",
    "# 3. Access a non-existent key using [] (KeyError)\n",
    "try:\n",
    "    cloud_tools = learning_paths[\"Cloud Computing\"]\n",
    "except KeyError as e:\n",
    "    print(f\"3. Error accessing 'Cloud Computing' with []: {e}\")\n",
    "# Output: Error accessing 'Cloud Computing' with []: 'Cloud Computing'\n",
    "\n",
    "# 4. Access the tools for Data Engineering using get()\n",
    "de_tools = learning_paths.get(\"Data Engineering\")\n",
    "print(f\"4. Data Engineering tools (get): {de_tools}\") # Output: ['ADF', 'ADLS', 'Databricks']\n",
    "\n",
    "# 5. Access non-existent key using get() (returns None)\n",
    "cloud_tools_get = learning_paths.get(\"Cloud Computing\")\n",
    "print(f\"5. Cloud Computing tools (get): {cloud_tools_get}\") # Output: None\n",
    "\n",
    "# 6. Access non-existent key using get() with a default value\n",
    "ml_tools_get = learning_paths.get(\"Machine Learning\", []) # Default to empty list\n",
    "print(f\"6. Machine Learning tools (get with default): {ml_tools_get}\") # Output: []\n",
    "\n",
    "# 7. Add a new learning path (Azure DevOps)\n",
    "learning_paths[\"Azure DevOps\"] = [\"GitHub\", \"Pipelines\", \"Terraform\"]\n",
    "print(f\"7. Learning paths after adding ADO: {learning_paths}\")\n",
    "\n",
    "# 8. Update the tools for Data Science\n",
    "learning_paths[\"Data Science\"] = [\"Python\", \"Pandas\", \"ML\", \"Visualization\"]\n",
    "print(f\"8. Learning paths after updating DS: {learning_paths}\")\n",
    "\n",
    "# 9. Add a new student rating\n",
    "student_ratings[\"Charlie\"] = 4.2\n",
    "print(f\"9. Student ratings after adding Charlie: {student_ratings}\")\n",
    "# Output: {'Alice': 4.5, 'Bob': 4.8, 'Charlie': 4.2}\n",
    "\n",
    "# 10. Update Bob's rating\n",
    "student_ratings[\"Bob\"] = 4.9\n",
    "print(f\"10. Student ratings after updating Bob: {student_ratings}\")\n",
    "# Output: {'Alice': 4.5, 'Bob': 4.9, 'Charlie': 4.2}\n",
    "```\n",
    "\n",
    "#### b) `pop(key[, default])`\n",
    "**Explanation:** Removes the item with the specified `key` and returns its value. If the key is not found, it raises a `KeyError`, unless a `default` value is provided, in which case the default value is returned. Modifies the dictionary in-place.\n",
    "**Examples:**\n",
    "```python\n",
    "learning_paths = {\n",
    "    \"Data Science\": [\"Python\", \"Pandas\", \"ML\"],\n",
    "    \"Data Engineering\": [\"ADF\", \"ADLS\", \"Databricks\"],\n",
    "    \"Azure DevOps\": [\"GitHub\", \"Pipelines\", \"Terraform\"]\n",
    "}\n",
    "\n",
    "# 1. Pop the \"Data Engineering\" path and get its tools\n",
    "de_tools = learning_paths.pop(\"Data Engineering\")\n",
    "print(f\"1. Popped DE tools: {de_tools}\")\n",
    "print(f\"   Dict after pop: {learning_paths}\")\n",
    "# Output:\n",
    "# Popped DE tools: ['ADF', 'ADLS', 'Databricks']\n",
    "# Dict after pop: {'Data Science': ['Python', 'Pandas', 'ML'], 'Azure DevOps': ['GitHub', 'Pipelines', 'Terraform']}\n",
    "\n",
    "# 2. Pop \"Data Science\"\n",
    "ds_tools = learning_paths.pop(\"Data Science\")\n",
    "print(f\"2. Popped DS tools: {ds_tools}\")\n",
    "print(f\"   Dict after pop: {learning_paths}\")\n",
    "# Output:\n",
    "# Popped DS tools: ['Python', 'Pandas', 'ML']\n",
    "# Dict after pop: {'Azure DevOps': ['GitHub', 'Pipelines', 'Terraform']}\n",
    "\n",
    "# 3. Attempt to pop \"Data Science\" again (KeyError)\n",
    "try:\n",
    "    learning_paths.pop(\"Data Science\")\n",
    "except KeyError as e:\n",
    "    print(f\"3. Error popping 'Data Science' again: {e}\")\n",
    "# Output: Error popping 'Data Science' again: 'Data Science'\n",
    "\n",
    "# 4. Pop a non-existent key (\"Cloud\") with a default value\n",
    "cloud_tools = learning_paths.pop(\"Cloud Computing\", \"Path not found\")\n",
    "print(f\"4. Popping 'Cloud Computing' with default: {cloud_tools}\")\n",
    "print(f\"   Dict state unchanged: {learning_paths}\")\n",
    "# Output:\n",
    "# Popping 'Cloud Computing' with default: Path not found\n",
    "# Dict state unchanged: {'Azure DevOps': ['GitHub', 'Pipelines', 'Terraform']}\n",
    "\n",
    "# 5. Pop the last remaining path (\"Azure DevOps\")\n",
    "ado_tools = learning_paths.pop(\"Azure DevOps\")\n",
    "print(f\"5. Popped ADO tools: {ado_tools}\")\n",
    "print(f\"   Dict after pop: {learning_paths}\")\n",
    "# Output:\n",
    "# Popped ADO tools: ['GitHub', 'Pipelines', 'Terraform']\n",
    "# Dict after pop: {}\n",
    "\n",
    "# 6. Attempt to pop from an empty dictionary (KeyError)\n",
    "try:\n",
    "    learning_paths.pop(\"Anything\")\n",
    "except KeyError as e:\n",
    "    print(f\"6. Error popping from empty dict: {e}\")\n",
    "# Output: Error popping from empty dict: 'Anything'\n",
    "\n",
    "# 7. Pop from empty dict with default value\n",
    "result = learning_paths.pop(\"Anything\", None)\n",
    "print(f\"7. Popping from empty dict with default: {result}\") # Output: None\n",
    "\n",
    "# 8. Pop based on a variable key\n",
    "path_to_remove = \"Data Engineering\" # Assume dict was reset\n",
    "learning_paths = {\"Data Science\": [], \"Data Engineering\": []}\n",
    "if path_to_remove in learning_paths:\n",
    "    removed_value = learning_paths.pop(path_to_remove)\n",
    "    print(f\"8. Removed value for '{path_to_remove}': {removed_value}\")\n",
    "else:\n",
    "    print(f\"8. Path '{path_to_remove}' not found.\")\n",
    "# Output: Removed value for 'Data Engineering': []\n",
    "\n",
    "# 9. Pop an item and immediately use its value\n",
    "student_ratings = {\"Alice\": 4.5, \"Bob\": 4.8}\n",
    "bob_rating = student_ratings.pop(\"Bob\")\n",
    "print(f\"9. Bob's rating was {bob_rating}, remaining: {student_ratings}\")\n",
    "# Output: Bob's rating was 4.8, remaining: {'Alice': 4.5}\n",
    "\n",
    "# 10. Using pop to iterate and remove (less common than iterating over keys/items)\n",
    "ratings_copy = {\"Alice\": 4.5, \"Bob\": 4.8}.copy()\n",
    "keys_to_pop = list(ratings_copy.keys()) # Need to copy keys first\n",
    "print(\"10. Popping based on key list:\")\n",
    "for key in keys_to_pop:\n",
    "    value = ratings_copy.pop(key)\n",
    "    print(f\"    Popped {key}: {value}\")\n",
    "print(f\"    Dict after loop: {ratings_copy}\")\n",
    "# Output:\n",
    "# Popping based on key list:\n",
    "#     Popped Alice: 4.5\n",
    "#     Popped Bob: 4.8\n",
    "#     Dict after loop: {}\n",
    "```\n",
    "\n",
    "#### c) `popitem()`\n",
    "**Explanation:** Removes and returns an arbitrary (key, value) pair as a tuple. In Python 3.7+, it follows LIFO (Last-In, First-Out) order - removing the most recently added item. Raises a `KeyError` if the dictionary is empty. Modifies the dictionary in-place.\n",
    "**Examples:**\n",
    "```python\n",
    "learning_paths = {\n",
    "    \"Data Science\": [\"Python\", \"Pandas\", \"ML\"],\n",
    "    \"Data Engineering\": [\"ADF\", \"ADLS\", \"Databricks\"],\n",
    "    \"Azure DevOps\": [\"GitHub\", \"Pipelines\", \"Terraform\"] # Last added\n",
    "}\n",
    "\n",
    "# 1. Pop the last inserted item (LIFO in Python 3.7+)\n",
    "key, value = learning_paths.popitem()\n",
    "print(f\"1. Popped item: ({key}: {value})\")\n",
    "print(f\"   Dict after popitem: {learning_paths}\")\n",
    "# Output (Python 3.7+):\n",
    "# Popped item: (Azure DevOps: ['GitHub', 'Pipelines', 'Terraform'])\n",
    "# Dict after popitem: {'Data Science': ['Python', 'Pandas', 'ML'], 'Data Engineering': ['ADF', 'ADLS', 'Databricks']}\n",
    "\n",
    "# 2. Pop the next last inserted item\n",
    "key, value = learning_paths.popitem()\n",
    "print(f\"2. Popped item: ({key}: {value})\")\n",
    "print(f\"   Dict after popitem: {learning_paths}\")\n",
    "# Output (Python 3.7+):\n",
    "# Popped item: (Data Engineering: ['ADF', 'ADLS', 'Databricks'])\n",
    "# Dict after popitem: {'Data Science': ['Python', 'Pandas', 'ML']}\n",
    "\n",
    "# 3. Pop the remaining item\n",
    "key, value = learning_paths.popitem()\n",
    "print(f\"3. Popped item: ({key}: {value})\")\n",
    "print(f\"   Dict after popitem: {learning_paths}\")\n",
    "# Output (Python 3.7+):\n",
    "# Popped item: (Data Science: ['Python', 'Pandas', 'ML'])\n",
    "# Dict after popitem: {}\n",
    "\n",
    "# 4. Attempt to popitem from an empty dictionary (KeyError)\n",
    "try:\n",
    "    learning_paths.popitem()\n",
    "except KeyError as e:\n",
    "    print(f\"4. Error popping item from empty dict: {e}\")\n",
    "# Output: Error popping item from empty dict: 'popitem(): dictionary is empty'\n",
    "\n",
    "# 5. Use popitem in a loop to process all items (LIFO order in 3.7+)\n",
    "paths_copy = { \"DS\": [], \"DE\": [], \"ADO\": [] }.copy()\n",
    "print(\"5. Processing paths using popitem (LIFO):\")\n",
    "while paths_copy: # While dict is not empty\n",
    "    path, tools = paths_copy.popitem()\n",
    "    print(f\"   - Processing: {path}\")\n",
    "# Output (Python 3.7+):\n",
    "# Processing paths using popitem (LIFO):\n",
    "#    - Processing: ADO\n",
    "#    - Processing: DE\n",
    "#    - Processing: DS\n",
    "\n",
    "# 6. Popitem from a dict with integer keys\n",
    "num_dict = {1: 'a', 2: 'b', 3: 'c'}\n",
    "k, v = num_dict.popitem()\n",
    "print(f\"6. Popped item from num_dict: ({k}: {v}), Remaining: {num_dict}\")\n",
    "# Output: Popped item from num_dict: (3: c), Remaining: {1: 'a', 2: 'b'}\n",
    "\n",
    "# 7. Popitem from a dict with tuple keys\n",
    "tuple_key_dict = {(0,0): \"Origin\", (1,1): \"Point A\"}\n",
    "k, v = tuple_key_dict.popitem()\n",
    "print(f\"7. Popped item from tuple_key_dict: ({k}: {v}), Remaining: {tuple_key_dict}\")\n",
    "# Output: Popped item from tuple_key_dict: ((1, 1): Point A), Remaining: {(0, 0): 'Origin'}\n",
    "\n",
    "# 8. Dict created in different order\n",
    "paths_alt_order = {}\n",
    "paths_alt_order[\"DS\"] = []\n",
    "paths_alt_order[\"DE\"] = []\n",
    "paths_alt_order[\"ADO\"] = [] # ADO added last\n",
    "k, v = paths_alt_order.popitem()\n",
    "print(f\"8. Popped from alt order dict: ({k}: {v})\") # Output: (ADO: [])\n",
    "\n",
    "# 9. Single item dictionary popitem\n",
    "single_item_dict = {\"Edukron\": \"Institute\"}\n",
    "k, v = single_item_dict.popitem()\n",
    "print(f\"9. Popped from single item dict: ({k}: {v}), Remaining: {single_item_dict}\")\n",
    "# Output: Popped from single item dict: (Edukron: Institute), Remaining: {}\n",
    "\n",
    "# 10. Storing popped key/value separately\n",
    "d = {'x': 1, 'y': 2}\n",
    "popped_pair = d.popitem()\n",
    "popped_key = popped_pair[0]\n",
    "popped_value = popped_pair[1]\n",
    "print(f\"10. Popped Key: {popped_key}, Popped Value: {popped_value}\") # Output: Popped Key: y, Popped Value: 2\n",
    "```\n",
    "\n",
    "#### d) `keys()`, `values()`, `items()`\n",
    "**Explanation:** These methods return \"view objects\" that provide dynamic views of the dictionary's keys, values, or key-value (item) pairs respectively. Views reflect changes made to the dictionary. They can be iterated over or converted to lists/tuples.\n",
    "**Examples:**\n",
    "```python\n",
    "learning_paths = {\n",
    "    \"Data Science\": [\"Python\", \"Pandas\", \"ML\"],\n",
    "    \"Data Engineering\": [\"ADF\", \"ADLS\", \"Databricks\"],\n",
    "    \"Azure DevOps\": [\"GitHub\", \"Pipelines\", \"Terraform\"]\n",
    "}\n",
    "\n",
    "# 1. Get a view of the keys\n",
    "path_keys = learning_paths.keys()\n",
    "print(f\"1. Keys view: {path_keys}\") # Output: dict_keys(['Data Science', 'Data Engineering', 'Azure DevOps'])\n",
    "# Convert to list\n",
    "path_key_list = list(path_keys)\n",
    "print(f\"   Keys as list: {path_key_list}\")\n",
    "\n",
    "# 2. Get a view of the values\n",
    "path_values = learning_paths.values()\n",
    "print(f\"\\n2. Values view: {path_values}\") # Output: dict_values([['Python', 'Pandas', 'ML'], [...], [...]])\n",
    "# Convert to list\n",
    "path_value_list = list(path_values)\n",
    "print(f\"   Values as list: {path_value_list}\")\n",
    "\n",
    "# 3. Get a view of the key-value items (tuples)\n",
    "path_items = learning_paths.items()\n",
    "print(f\"\\n3. Items view: {path_items}\") # Output: dict_items([('Data Science', [...]), ('Data Engineering', [...]), ...])\n",
    "# Convert to list\n",
    "path_item_list = list(path_items)\n",
    "print(f\"   Items as list: {path_item_list}\")\n",
    "\n",
    "# 4. Iterate directly over keys view\n",
    "print(\"\\n4. Iterating over keys:\")\n",
    "for path_name in learning_paths.keys():\n",
    "    print(f\"   - Path: {path_name}\")\n",
    "\n",
    "# 5. Iterate directly over values view\n",
    "print(\"\\n5. Iterating over values:\")\n",
    "for tool_list in learning_paths.values():\n",
    "    print(f\"   - Tools: {tool_list}\")\n",
    "\n",
    "# 6. Iterate directly over items view (unpacking key/value)\n",
    "print(\"\\n6. Iterating over items:\")\n",
    "for path_name, tool_list in learning_paths.items():\n",
    "    print(f\"   - Path: {path_name}, First tool: {tool_list[0] if tool_list else 'N/A'}\")\n",
    "\n",
    "# 7. Views reflect dictionary changes\n",
    "student_ratings = {\"Alice\": 4.5, \"Bob\": 4.8}\n",
    "ratings_view = student_ratings.values()\n",
    "print(f\"\\n7. Initial ratings view: {ratings_view}\") # Output: dict_values([4.5, 4.8])\n",
    "student_ratings[\"Charlie\"] = 4.2 # Modify the dictionary\n",
    "print(f\"   Ratings view after adding Charlie: {ratings_view}\") # Output: dict_values([4.5, 4.8, 4.2])\n",
    "\n",
    "# 8. Check membership in keys view (efficient)\n",
    "has_ds = \"Data Science\" in learning_paths.keys() # More explicit than `in learning_paths`\n",
    "print(f\"\\n8. Is 'Data Science' in keys view? {has_ds}\") # Output: True\n",
    "\n",
    "# 9. Check membership in values view (less common/efficient)\n",
    "# This checks if the exact list object/value exists as a value\n",
    "is_list_present = [\"Python\", \"Pandas\", \"ML\"] in learning_paths.values()\n",
    "print(f\"9. Is ['Python', 'Pandas', 'ML'] in values view? {is_list_present}\") # Output: True\n",
    "\n",
    "# 10. Using items to create a list of formatted strings\n",
    "description_list = [f\"{k}: {v[0]}...\" for k, v in learning_paths.items()]\n",
    "print(f\"\\n10. List from items view: {description_list}\")\n",
    "# Output: ['Data Science: Python...', 'Data Engineering: ADF...', 'Azure DevOps: GitHub...']\n",
    "```\n",
    "*(Also supports `len()`, `in` (checks keys), `not in`)*\n",
    "\n",
    "---\n",
    "\n",
    "##  Boolean Type (`bool`)\n",
    "\n",
    "**Explanation:** Represents one of two truth values: `True` or `False`. Booleans are fundamental for conditional logic (`if`, `while`), comparisons, and representing status flags. They are a subclass of integers where `True` corresponds to `1` and `False` corresponds to `0`.\n",
    "\n",
    "Your examples:\n",
    "*   `is_data_science_active = True`\n",
    "*   `is_azure_devops_course_full = False`\n",
    "*   `is_python_essential = True`\n",
    "\n",
    "**Common Operations & Usage:**\n",
    "\n",
    "#### a) Literals and Comparisons\n",
    "**Explanation:** Using the literal values `True` and `False`, and generating booleans through comparison operators (`==`, `!=`, `<`, `<=`, `>`, `>=`).\n",
    "**Examples:**\n",
    "```python\n",
    "total_students = 1500\n",
    "capacity = 1500\n",
    "average_rating = 4.8\n",
    "min_rating = 4.5\n",
    "\n",
    "# 1. Boolean literal True\n",
    "is_data_science_active = True\n",
    "print(f\"1. DS Active Status: {is_data_science_active}\") # Output: True\n",
    "\n",
    "# 2. Boolean literal False\n",
    "is_azure_devops_course_full = False\n",
    "print(f\"2. ADO Full Status: {is_azure_devops_course_full}\") # Output: False\n",
    "\n",
    "# 3. Equality check (==)\n",
    "at_capacity = (total_students == capacity)\n",
    "print(f\"3. Is course at capacity? {at_capacity}\") # Output: True\n",
    "\n",
    "# 4. Inequality check (!=)\n",
    "has_vacancies = (total_students != capacity)\n",
    "print(f\"4. Are there vacancies? {has_vacancies}\") # Output: False\n",
    "\n",
    "# 5. Greater than check (>)\n",
    "rating_is_good = (average_rating > min_rating)\n",
    "print(f\"5. Is average rating good? {rating_is_good}\") # Output: True\n",
    "\n",
    "# 6. Less than check (<)\n",
    "needs_improvement = (average_rating < 4.0)\n",
    "print(f\"6. Does rating need improvement? {needs_improvement}\") # Output: False\n",
    "\n",
    "# 7. Greater than or equal check (>=)\n",
    "meets_min_rating = (average_rating >= min_rating)\n",
    "print(f\"7. Does rating meet minimum? {meets_min_rating}\") # Output: True\n",
    "\n",
    "# 8. Less than or equal check (<=)\n",
    "enrollment_below_target = (total_students <= 2000)\n",
    "print(f\"8. Is enrollment <= 2000? {enrollment_below_target}\") # Output: True\n",
    "\n",
    "# 9. Comparing strings\n",
    "institute = \"Edukron\"\n",
    "is_edukron = (institute == \"Edukron\")\n",
    "print(f\"9. Is institute name 'Edukron'? {is_edukron}\") # Output: True\n",
    "\n",
    "# 10. Comparing booleans themselves\n",
    "are_flags_equal = (is_data_science_active == is_python_essential) # True == True\n",
    "print(f\"10. Are DS active and Python essential flags equal? {are_flags_equal}\") # Output: True\n",
    "```\n",
    "\n",
    "#### b) Boolean Operators (`and`, `or`, `not`)\n",
    "**Explanation:** Combining boolean values.\n",
    "*   `x and y`: `True` only if both `x` and `y` are `True`. Short-circuits (if `x` is `False`, `y` is not evaluated).\n",
    "*   `x or y`: `True` if at least one of `x` or `y` is `True`. Short-circuits (if `x` is `True`, `y` is not evaluated).\n",
    "*   `not x`: `True` if `x` is `False`, `False` if `x` is `True`.\n",
    "**Examples:**\n",
    "```python\n",
    "is_data_science_active = True\n",
    "is_azure_devops_course_full = False\n",
    "average_rating = 4.8\n",
    "min_rating = 4.5\n",
    "has_prerequisites = True\n",
    "\n",
    "# 1. AND: Check if DS is active AND rating is good\n",
    "can_recommend_ds = is_data_science_active and (average_rating > min_rating)\n",
    "print(f\"1. Can recommend Data Science course? {can_recommend_ds}\") # Output: True (True and True)\n",
    "\n",
    "# 2. AND: Check if ADO is full AND rating is good (Short-circuits)\n",
    "should_waitlist_ado = is_azure_devops_course_full and (average_rating > min_rating)\n",
    "print(f\"2. Should waitlist for Azure DevOps? {should_waitlist_ado}\") # Output: False (False and ...)\n",
    "\n",
    "# 3. OR: Check if DS is active OR ADO is full\n",
    "show_course_status = is_data_science_active or is_azure_devops_course_full\n",
    "print(f\"3. Show status (DS active or ADO full)? {show_course_status}\") # Output: True (True or False)\n",
    "\n",
    "# 4. OR: Check if rating is low OR ADO is full (Short-circuits)\n",
    "needs_attention = (average_rating < 4.0) or is_azure_devops_course_full\n",
    "print(f\"4. Does Edukron need attention (low rating or full ADO)? {needs_attention}\") # Output: False (False or False)\n",
    "\n",
    "# 5. NOT: Check if ADO course is NOT full\n",
    "can_enroll_ado = not is_azure_devops_course_full\n",
    "print(f\"5. Can enroll in Azure DevOps? {can_enroll_ado}\") # Output: True (not False)\n",
    "\n",
    "# 6. NOT: Check if DS course is NOT active\n",
    "is_ds_inactive = not is_data_science_active\n",
    "print(f\"6. Is Data Science course inactive? {is_ds_inactive}\") # Output: False (not True)\n",
    "\n",
    "# 7. Combining operators: Active course with good rating and prerequisites met\n",
    "eligible_to_start = is_data_science_active and (average_rating >= min_rating) and has_prerequisites\n",
    "print(f\"7. Eligible to start Data Science? {eligible_to_start}\") # Output: True (True and True and True)\n",
    "\n",
    "# 8. Combining operators: Not full OR has prerequisites\n",
    "can_consider = (not is_azure_devops_course_full) or has_prerequisites\n",
    "print(f\"8. Can consider ADO (not full or has prereqs)? {can_consider}\") # Output: True (True or True)\n",
    "\n",
    "# 9. Complex combination with parentheses\n",
    "# Active AND (Good rating OR Not Full)\n",
    "complex_check = is_data_science_active and ((average_rating > 4.7) or (not is_azure_devops_course_full))\n",
    "print(f\"9. Complex Check Result: {complex_check}\") # Output: True (True and (True or True))\n",
    "\n",
    "# 10. Using boolean variable directly with 'not'\n",
    "is_python_essential = True\n",
    "is_optional = not is_python_essential\n",
    "print(f\"10. Is Python optional? {is_optional}\") # Output: False\n",
    "```\n",
    "\n",
    "#### c) `bool()` Constructor and Truth Value Testing\n",
    "**Explanation:** The `bool()` function can convert other types to `True` or `False`. In Python, certain values are considered \"falsy\" (evaluate to `False` in a boolean context), while most others are \"truthy\" (evaluate to `True`).\n",
    "*   **Falsy values:** `None`, `False`, zero of any numeric type (`0`, `0.0`, `0j`), empty sequences (`''`, `()`, `[]`), empty mappings (`{}`), empty sets (`set()`).\n",
    "*   **Truthy values:** Everything else, including non-empty sequences/mappings, non-zero numbers, `True`.\n",
    "**Examples:**\n",
    "```python\n",
    "total_students = 1500\n",
    "average_rating = 4.8\n",
    "course_list = [\"Data Science\", \"Data Engineering\"]\n",
    "empty_list = []\n",
    "student_name = \"Alice\"\n",
    "no_name = \"\"\n",
    "zero_count = 0\n",
    "\n",
    "# 1. bool() of a non-zero integer\n",
    "bool_students = bool(total_students)\n",
    "print(f\"1. bool({total_students}): {bool_students}\") # Output: True\n",
    "\n",
    "# 2. bool() of zero\n",
    "bool_zero = bool(zero_count)\n",
    "print(f\"2. bool({zero_count}): {bool_zero}\") # Output: False\n",
    "\n",
    "# 3. bool() of a non-empty list\n",
    "bool_courses = bool(course_list)\n",
    "print(f\"3. bool(course_list): {bool_courses}\") # Output: True\n",
    "\n",
    "# 4. bool() of an empty list\n",
    "bool_empty = bool(empty_list)\n",
    "print(f\"4. bool(empty_list): {bool_empty}\") # Output: False\n",
    "\n",
    "# 5. bool() of a non-empty string\n",
    "bool_name = bool(student_name)\n",
    "print(f\"5. bool('{student_name}'): {bool_name}\") # Output: True\n",
    "\n",
    "# 6. bool() of an empty string\n",
    "bool_no_name = bool(no_name)\n",
    "print(f\"6. bool(''): {bool_no_name}\") # Output: False\n",
    "\n",
    "# 7. bool() of None\n",
    "bool_none = bool(None)\n",
    "print(f\"7. bool(None): {bool_none}\") # Output: False\n",
    "\n",
    "# 8. Truth value testing in 'if' (non-empty list is True)\n",
    "if course_list:\n",
    "    print(\"8. Course list is not empty (truthy).\")\n",
    "else:\n",
    "    print(\"8. Course list is empty (falsy).\")\n",
    "# Output: Course list is not empty (truthy).\n",
    "\n",
    "# 9. Truth value testing in 'if' (empty list is False)\n",
    "if empty_list:\n",
    "    print(\"9. This won't print (empty list is falsy).\")\n",
    "else:\n",
    "    print(\"9. Empty list evaluated as False.\")\n",
    "# Output: Empty list evaluated as False.\n",
    "\n",
    "# 10. Truth value testing with 'or' default pattern\n",
    "# Get student name, or use \"Guest\" if name is empty (falsy)\n",
    "current_user = no_name or \"Guest\"\n",
    "print(f\"10. Current user (empty name): {current_user}\") # Output: Guest\n",
    "current_user = student_name or \"Guest\"\n",
    "print(f\"    Current user (non-empty name): {current_user}\") # Output: Alice\n",
    "```\n",
    "\n",
    "---\n",
    "\n",
    "##  Binary Types\n",
    "\n",
    "Binary types are used to represent data in its raw byte form. This is crucial for handling binary files (like images, executables), network communication, and low-level data manipulation.\n",
    "\n",
    "### 1. `bytes`\n",
    "\n",
    "**Explanation:** Represents an **immutable** sequence of bytes (integers between 0 and 255). Created using a `b` prefix before a string literal (e.g., `b'hello'`) or the `bytes()` constructor. String literals within `b''` must typically be ASCII-compatible; for other encodings, use `.encode()`.\n",
    "\n",
    "Your example:\n",
    "*   `project_name = bytes(\"Edukron Projects\", encoding=\"utf-8\")` (or `b'Edukron Projects'` if pure ASCII)\n",
    "\n",
    "**Common Operations & Methods:**\n",
    "\n",
    "Bytes objects share many sequence operations with strings and lists (indexing, slicing, `len()`, `+`, `*`, `in`) but operate on bytes. Key methods include `decode()` and `hex()`.\n",
    "\n",
    "#### a) Creation, Indexing, Slicing\n",
    "**Explanation:** Creating bytes objects and accessing individual bytes or sequences of bytes. Individual bytes are returned as integers.\n",
    "**Examples:**\n",
    "```python\n",
    "# 1. Create bytes using literal (ASCII only)\n",
    "b_literal = b'Edukron'\n",
    "print(f\"1. Bytes literal: {b_literal}\") # Output: b'Edukron'\n",
    "\n",
    "# 2. Create bytes from string using encode() (recommended for non-ASCII)\n",
    "project_name_str = \"Edukron Projects\"\n",
    "project_name_bytes = project_name_str.encode('utf-8')\n",
    "print(f\"2. Bytes from encoded string: {project_name_bytes}\") # Output: b'Edukron Projects\\xc2\\xa9'\n",
    "\n",
    "# 3. Create bytes from a list of integers (0-255)\n",
    "byte_list = [69, 100, 117, 107, 114, 111, 110] # ASCII for 'Edukron'\n",
    "b_from_list = bytes(byte_list)\n",
    "print(f\"3. Bytes from list of ints: {b_from_list}\") # Output: b'Edukron'\n",
    "\n",
    "# 4. Access the first byte (returns an integer)\n",
    "first_byte = b_literal[0]\n",
    "print(f\"4. First byte of {b_literal}: {first_byte} (ASCII: {chr(first_byte)})\") # Output: 69 (ASCII: E)\n",
    "\n",
    "# 5. Access the last byte\n",
    "last_byte = b_literal[-1]\n",
    "print(f\"5. Last byte of {b_literal}: {last_byte} (ASCII: {chr(last_byte)})\") # Output: 110 (ASCII: n)\n",
    "\n",
    "# 6. Get a slice of bytes (returns bytes object)\n",
    "slice_bytes = b_literal[0:3] # bytes at index 0, 1, 2\n",
    "print(f\"6. Slice [0:3] of {b_literal}: {slice_bytes}\") # Output: b'Edu'\n",
    "\n",
    "# 7. Get bytes from index 3 onwards\n",
    "slice_from_3 = b_literal[3:]\n",
    "print(f\"7. Slice [3:] of {b_literal}: {slice_from_3}\") # Output: b'kron'\n",
    "\n",
    "# 8. Create empty bytes object\n",
    "empty_bytes = b''\n",
    "print(f\"8. Empty bytes: {empty_bytes}\")\n",
    "\n",
    "# 9. Create bytes object of a specific size, initialized to null bytes\n",
    "null_bytes = bytes(5) # 5 null bytes (\\x00)\n",
    "print(f\"9. Null bytes of size 5: {null_bytes}\") # Output: b'\\x00\\x00\\x00\\x00\\x00'\n",
    "\n",
    "# 10. Accessing bytes in UTF-8 sequence\n",
    "# project_name_bytes = b'Edukron Projects\\xc2\\xa9'\n",
    "copyright_byte_1 = project_name_bytes[-2]\n",
    "copyright_byte_2 = project_name_bytes[-1]\n",
    "print(f\"10. UTF-8 bytes for : {copyright_byte_1}, {copyright_byte_2}\") # Output: 194, 169 (0xC2, 0xA9)\n",
    "```\n",
    "\n",
    "#### b) `decode(encoding='utf-8', errors='strict')`\n",
    "**Explanation:** Converts the `bytes` object back into a `str`, using the specified `encoding`. `errors` handles cases where bytes are invalid for the encoding ('strict' raises error, 'ignore' drops them, 'replace' inserts replacement char).\n",
    "**Examples:**\n",
    "```python\n",
    "project_name_bytes = b'Edukron Projects\\xc2\\xa9'\n",
    "b_literal = b'Edukron'\n",
    "invalid_utf8 = b'Edukron\\xffInvalid' # \\xff is not valid UTF-8 start byte\n",
    "\n",
    "# 1. Decode UTF-8 bytes back to string\n",
    "decoded_project_name = project_name_bytes.decode('utf-8')\n",
    "print(f\"1. Decoded project name (utf-8): '{decoded_project_name}'\") # Output: 'Edukron Projects'\n",
    "\n",
    "# 2. Decode simple ASCII bytes (utf-8 is compatible)\n",
    "decoded_literal = b_literal.decode('utf-8')\n",
    "print(f\"2. Decoded ASCII literal (utf-8): '{decoded_literal}'\") # Output: 'Edukron'\n",
    "\n",
    "# 3. Decode simple ASCII bytes using 'ascii' encoding\n",
    "decoded_ascii = b_literal.decode('ascii')\n",
    "print(f\"3. Decoded ASCII literal (ascii): '{decoded_ascii}'\") # Output: 'Edukron'\n",
    "\n",
    "# 4. Attempt to decode non-ASCII using 'ascii' (UnicodeDecodeError)\n",
    "try:\n",
    "    project_name_bytes.decode('ascii')\n",
    "except UnicodeDecodeError as e:\n",
    "    print(f\"4. Error decoding non-ASCII with 'ascii': {e}\")\n",
    "# Output: Error ... 'ascii' codec can't decode byte 0xc2 in position 16...\n",
    "\n",
    "# 5. Decode invalid UTF-8 with errors='strict' (UnicodeDecodeError)\n",
    "try:\n",
    "    invalid_utf8.decode('utf-8', errors='strict')\n",
    "except UnicodeDecodeError as e:\n",
    "    print(f\"5. Error decoding invalid UTF-8 (strict): {e}\")\n",
    "# Output: Error ... 'utf-8' codec can't decode byte 0xff in position 7...\n",
    "\n",
    "# 6. Decode invalid UTF-8 with errors='ignore'\n",
    "decoded_ignore = invalid_utf8.decode('utf-8', errors='ignore')\n",
    "print(f\"6. Decoded invalid UTF-8 (ignore): '{decoded_ignore}'\") # Output: 'EdukronInvalid'\n",
    "\n",
    "# 7. Decode invalid UTF-8 with errors='replace'\n",
    "decoded_replace = invalid_utf8.decode('utf-8', errors='replace')\n",
    "print(f\"7. Decoded invalid UTF-8 (replace): '{decoded_replace}'\") # Output: 'EdukronInvalid' ( is replacement)\n",
    "\n",
    "# 8. Decode bytes representing Latin-1 (ISO-8859-1)\n",
    "# Example: The  symbol is 0xA3 in Latin-1\n",
    "pound_latin1 = b'\\xA3100'\n",
    "decoded_pound = pound_latin1.decode('latin-1')\n",
    "print(f\"8. Decoded Latin-1 bytes: '{decoded_pound}'\") # Output: '100'\n",
    "\n",
    "# 9. Decode empty bytes string\n",
    "decoded_empty = b''.decode('utf-8')\n",
    "print(f\"9. Decoded empty bytes: '{decoded_empty}'\") # Output: ''\n",
    "\n",
    "# 10. Decode bytes created from list\n",
    "b_from_list = bytes([69, 100, 117]) # Edu\n",
    "decoded_list_bytes = b_from_list.decode('ascii')\n",
    "print(f\"10. Decoded bytes from list: '{decoded_list_bytes}'\") # Output: 'Edu'\n",
    "```\n",
    "\n",
    "#### c) Other Common Methods (`startswith`, `endswith`, `find`, `replace`, `hex`)\n",
    "**Explanation:** Bytes objects have many methods analogous to strings, but operating on bytes. `hex()` provides hexadecimal representation.\n",
    "**Examples:**\n",
    "```python\n",
    "header = b'EDUKRON_DATA_v1.0'\n",
    "data = b'\\x01\\x02\\x03\\x04Edukron\\x05\\x06'\n",
    "footer = b'END_DATA'\n",
    "\n",
    "# 1. Check if header starts with b'EDUKRON'\n",
    "starts_edu = header.startswith(b'EDUKRON')\n",
    "print(f\"1. Header starts with b'EDUKRON'? {starts_edu}\") # Output: True\n",
    "\n",
    "# 2. Check if header ends with b'v1.0'\n",
    "ends_v1 = header.endswith(b'v1.0')\n",
    "print(f\"2. Header ends with b'v1.0'? {ends_v1}\") # Output: True\n",
    "\n",
    "# 3. Find the position of b'DATA' in header\n",
    "data_pos = header.find(b'DATA')\n",
    "print(f\"3. Position of b'DATA' in header: {data_pos}\") # Output: 8\n",
    "\n",
    "# 4. Find position of b'Edukron' in data bytes\n",
    "edu_pos = data.find(b'Edukron')\n",
    "print(f\"4. Position of b'Edukron' in data: {edu_pos}\") # Output: 4\n",
    "\n",
    "# 5. Replace b'v1.0' with b'v1.1' in header (creates new bytes)\n",
    "new_header = header.replace(b'v1.0', b'v1.1')\n",
    "print(f\"5. New header after replace: {new_header}\") # Output: b'EDUKRON_DATA_v1.1'\n",
    "\n",
    "# 6. Get hexadecimal representation of data bytes\n",
    "hex_data = data.hex()\n",
    "print(f\"6. Hex representation of data: {hex_data}\")\n",
    "# Output: 010203044564756b726f6e0506 (E d u k r o n)\n",
    "\n",
    "# 7. Get hex representation with a separator\n",
    "hex_data_sep = data.hex(sep=':')\n",
    "print(f\"7. Hex representation with separator: {hex_data_sep}\")\n",
    "# Output: 01:02:03:04:45:64:75:6b:72:6f:6e:05:06\n",
    "\n",
    "# 8. Concatenate bytes using +\n",
    "full_message = header + data + footer\n",
    "print(f\"8. Full message (concatenated): {full_message[:20]}...\") # Show first 20 bytes\n",
    "# Output: b'EDUKRON_DATA_v1.0\\x01\\x02\\x03\\x04'...\n",
    "\n",
    "# 9. Check for membership using 'in'\n",
    "is_edu_in_data = b'Edukron' in data\n",
    "print(f\"9. Is b'Edukron' in data? {is_edu_in_data}\") # Output: True\n",
    "\n",
    "# 10. Get length (number of bytes)\n",
    "data_len = len(data)\n",
    "print(f\"10. Length of data bytes: {data_len}\") # Output: 13\n",
    "```\n",
    "\n",
    "---\n",
    "\n",
    "### 2. `bytearray`\n",
    "\n",
    "**Explanation:** Represents a **mutable** sequence of bytes (integers between 0 and 255). It has most of the same methods as `bytes`, but also supports in-place modification of elements and methods like `append`, `extend`, `insert`, `remove` (similar to lists, but operating on bytes). Created using the `bytearray()` constructor.\n",
    "\n",
    "Your example:\n",
    "*   `course_stats = bytearray([10, 20, 30, 40])`\n",
    "\n",
    "**Common Operations & Methods:**\n",
    "\n",
    "#### a) Creation and Mutability\n",
    "**Explanation:** Creating bytearrays and modifying their contents directly by index or using list-like methods.\n",
    "**Examples:**\n",
    "```python\n",
    "# 1. Create from a list of integers\n",
    "course_stats = bytearray([10, 20, 30, 40])\n",
    "print(f\"1. Initial bytearray: {course_stats}\") # Output: bytearray(b'\\n\\x14\\x1e(')\n",
    "\n",
    "# 2. Create from a bytes object\n",
    "b_literal = b'Edukron'\n",
    "ba_from_bytes = bytearray(b_literal)\n",
    "print(f\"2. Bytearray from bytes: {ba_from_bytes}\") # Output: bytearray(b'Edukron')\n",
    "\n",
    "# 3. Create from a string with encoding\n",
    "ba_from_str = bytearray(\"Header\", 'utf-8')\n",
    "print(f\"3. Bytearray from string: {ba_from_str}\") # Output: bytearray(b'Header')\n",
    "\n",
    "# 4. Create an empty bytearray\n",
    "empty_ba = bytearray()\n",
    "print(f\"4. Empty bytearray: {empty_ba}\")\n",
    "\n",
    "# 5. Create zero-filled bytearray of specific size\n",
    "zero_ba = bytearray(5)\n",
    "print(f\"5. Zero-filled bytearray: {zero_ba}\") # Output: bytearray(b'\\x00\\x00\\x00\\x00\\x00')\n",
    "\n",
    "# 6. Modify a byte at a specific index\n",
    "print(f\"6. Course stats before modify: {course_stats}\")\n",
    "course_stats[0] = 15 # Change first byte from 10 to 15\n",
    "print(f\"   Course stats after modify index 0: {course_stats}\")\n",
    "# Output: bytearray(b'\\x0f\\x14\\x1e(')\n",
    "\n",
    "# 7. Modify a slice (must assign iterable of bytes/ints)\n",
    "print(f\"7. Course stats before slice assign: {course_stats}\")\n",
    "course_stats[1:3] = [25, 35] # Replace bytes at index 1, 2\n",
    "# Or use bytes: course_stats[1:3] = b'\\x19\\x23'\n",
    "print(f\"   Course stats after slice assign: {course_stats}\")\n",
    "# Output: bytearray(b'\\x0f\\x19#(') (Note: 35 is '#')\n",
    "\n",
    "# 8. Append a single byte (integer 0-255)\n",
    "course_stats.append(50)\n",
    "print(f\"8. Course stats after append: {course_stats}\")\n",
    "# Output: bytearray(b'\\x0f\\x19#(2') (Note: 50 is '2')\n",
    "\n",
    "# 9. Extend with an iterable of bytes/ints\n",
    "course_stats.extend([60, 70])\n",
    "# Or use bytes: course_stats.extend(b'<F')\n",
    "print(f\"9. Course stats after extend: {course_stats}\")\n",
    "# Output: bytearray(b'\\x0f\\x19#(2<F')\n",
    "\n",
    "# 10. Attempt to assign integer > 255 (ValueError)\n",
    "try:\n",
    "    course_stats[0] = 300\n",
    "except ValueError as e:\n",
    "    print(f\"10. Error assigning value > 255: {e}\")\n",
    "# Output: Error assigning value > 255: byte must be in range(0, 256)\n",
    "```\n",
    "*(Bytearray also supports `decode`, `hex`, `startswith`, `endswith`, `find`, `replace`, `pop`, `remove`, `insert`, `len`, `+`, `*`, `in` similar to `bytes` and `list`)*\n",
    "\n",
    "---\n",
    "\n",
    "### 3. `memoryview`\n",
    "\n",
    "**Explanation:** Creates a memory view object that allows accessing the internal data of another object (like `bytes`, `bytearray`) **without making a copy**. This is efficient for large data. If the underlying object is mutable (like `bytearray`), the memoryview can also be used to modify the data in-place.\n",
    "\n",
    "Your example:\n",
    "*   `memory_snapshot = memoryview(bytes(5))` (Creates a view of 5 null bytes)\n",
    "\n",
    "**Common Operations & Usage:**\n",
    "\n",
    "Memoryviews support indexing and slicing, which also return memoryviews (or single bytes as integers). They expose the data in various formats using methods like `tobytes()`, `tolist()`, and allow modification if the underlying object permits.\n",
    "\n",
    "#### a) Creation, Accessing Data, Slicing\n",
    "**Explanation:** Creating memoryviews from binary objects and accessing their data.\n",
    "**Examples:**\n",
    "```python\n",
    "data_bytes = b'Edukron Data'\n",
    "data_bytearray = bytearray(b'\\x01\\x02\\x03\\x04\\x05')\n",
    "\n",
    "# 1. Create memoryview from bytes\n",
    "mv_bytes = memoryview(data_bytes)\n",
    "print(f\"1. Memoryview from bytes: {mv_bytes}\") # Output: <memory at 0x...>\n",
    "\n",
    "# 2. Create memoryview from bytearray\n",
    "mv_bytearray = memoryview(data_bytearray)\n",
    "print(f\"2. Memoryview from bytearray: {mv_bytearray}\") # Output: <memory at 0x...>\n",
    "\n",
    "# 3. Access length via memoryview\n",
    "print(f\"3. Length via mv_bytes: {len(mv_bytes)}\") # Output: 12\n",
    "print(f\"   Length via mv_bytearray: {len(mv_bytearray)}\") # Output: 5\n",
    "\n",
    "# 4. Access individual byte (returns integer)\n",
    "first_byte = mv_bytes[0]\n",
    "print(f\"4. First byte via mv_bytes: {first_byte}\") # Output: 69 (ASCII 'E')\n",
    "\n",
    "# 5. Access slice (returns another memoryview)\n",
    "mv_slice = mv_bytes[8:12] # Slice for \"Data\"\n",
    "print(f\"5. Slice via mv_bytes: {mv_slice}\") # Output: <memory at 0x...>\n",
    "# Convert slice to bytes to see content\n",
    "print(f\"   Slice content: {mv_slice.tobytes()}\") # Output: b'Data'\n",
    "\n",
    "# 6. Convert memoryview back to bytes\n",
    "bytes_copy = mv_bytearray.tobytes()\n",
    "print(f\"6. mv_bytearray to bytes: {bytes_copy}\") # Output: b'\\x01\\x02\\x03\\x04\\x05'\n",
    "\n",
    "# 7. Convert memoryview to list of integers\n",
    "int_list = mv_bytearray.tolist()\n",
    "print(f\"7. mv_bytearray to list: {int_list}\") # Output: [1, 2, 3, 4, 5]\n",
    "\n",
    "# 8. Access underlying object\n",
    "underlying_obj = mv_bytes.obj\n",
    "print(f\"8. Underlying object of mv_bytes: {underlying_obj}\") # Output: b'Edukron Data'\n",
    "print(f\"   Is underlying object same as original? {underlying_obj is data_bytes}\") # Output: True\n",
    "\n",
    "# 9. Slicing the bytearray view\n",
    "mv_ba_slice = mv_bytearray[1:3] # View of bytes at index 1, 2\n",
    "print(f\"9. Slice via mv_bytearray: {mv_ba_slice.tolist()}\") # Output: [2, 3]\n",
    "\n",
    "# 10. Creating from the example memory_snapshot\n",
    "memory_snapshot = memoryview(bytes(5))\n",
    "print(f\"10. Memory snapshot content: {memory_snapshot.tolist()}\") # Output: [0, 0, 0, 0, 0]\n",
    "```\n",
    "\n",
    "#### b) Modifying Data via Memoryview (on Mutable Objects)\n",
    "**Explanation:** Using a memoryview to modify the underlying mutable object (like `bytearray`) without copying.\n",
    "**Examples:**\n",
    "```python\n",
    "mutable_data = bytearray(b'Hello Edukron')\n",
    "mv = memoryview(mutable_data)\n",
    "\n",
    "# 1. Modify a single byte via memoryview\n",
    "print(f\"1. Original bytearray: {mutable_data}\")\n",
    "mv[0] = ord('J') # Change 'H' (72) to 'J' (74)\n",
    "print(f\"   Bytearray after mv modification: {mutable_data}\")\n",
    "# Output: bytearray(b'Jello Edukron')\n",
    "\n",
    "# 2. Modify a slice via memoryview (assign iterable of ints)\n",
    "mv_slice = mv[6:13] # Slice for \"Edukron\"\n",
    "print(f\"2. Original slice bytes: {mv_slice.tobytes()}\")\n",
    "mv_slice[:] = b'PYTHON!' # Assign bytes of same length\n",
    "print(f\"   Bytearray after slice modification: {mutable_data}\")\n",
    "# Output: bytearray(b'Jello PYTHON!')\n",
    "\n",
    "# 3. Modify slice with different length (not allowed for memoryview slice assignment)\n",
    "try:\n",
    "    mv_slice = mv[0:5] # \"Jello\"\n",
    "    mv_slice[:] = b'Hi' # Assign shorter sequence\n",
    "except ValueError as e:\n",
    "    print(f\"3. Error assigning slice of different length: {e}\")\n",
    "# Output: Error ... memoryview assignment: lvalue and rvalue have different structures\n",
    "\n",
    "# NOTE: You *can* modify the underlying bytearray slice directly, which the view reflects\n",
    "mutable_data[0:5] = b'Hi' # Modify bytearray directly\n",
    "print(f\"3b. Bytearray after direct slice assign: {mutable_data}\") # Output: bytearray(b'Hi PYTHON!')\n",
    "print(f\"    Memoryview now reflects: {mv.tobytes()}\") # Output: b'Hi PYTHON!' (View is shorter now)\n",
    "\n",
    "# Reset for next examples\n",
    "mutable_data = bytearray(b'\\x00\\x01\\x02\\x03\\x04')\n",
    "mv = memoryview(mutable_data)\n",
    "\n",
    "# 4. Modify slice using list of integers\n",
    "mv[1:4] = [10, 20, 30] # Assign to indices 1, 2, 3\n",
    "print(f\"4. Bytearray after assigning list [10, 20, 30]: {mutable_data}\")\n",
    "# Output: bytearray(b'\\x00\\n\\x14\\x1e\\x04')\n",
    "\n",
    "# 5. Modify using memoryview slice index\n",
    "mv_s = mv[1:4] # View of bytes 10, 20, 30\n",
    "mv_s[0] = 11 # Modify first byte of the slice view (index 1 of original)\n",
    "print(f\"5. Bytearray after modifying slice view index 0: {mutable_data}\")\n",
    "# Output: bytearray(b'\\x00\\x0b\\x14\\x1e\\x04')\n",
    "\n",
    "# 6. Cannot modify memoryview of immutable object (bytes)\n",
    "immutable_data = b'Immutable'\n",
    "mv_imm = memoryview(immutable_data)\n",
    "try:\n",
    "    mv_imm[0] = ord('X')\n",
    "except TypeError as e:\n",
    "    print(f\"6. Error modifying memoryview of bytes: {e}\")\n",
    "# Output: Error ... cannot modify read-only memory\n",
    "\n",
    "# 7. Using memoryview for efficient partial reads/writes (conceptual)\n",
    "# Imagine reading into a large bytearray buffer\n",
    "buffer = bytearray(1024)\n",
    "mv_buffer = memoryview(buffer)\n",
    "# Read 100 bytes into start of buffer (simulated)\n",
    "# network_socket.readinto(mv_buffer[0:100])\n",
    "# Process header from first 10 bytes\n",
    "# process_header(mv_buffer[0:10].tobytes())\n",
    "print(\"7. Conceptual use for efficient I/O shown in comments.\")\n",
    "\n",
    "# 8. Releasing the memoryview (explicitly, though usually handled by GC)\n",
    "# Not a direct method, but letting go of references allows GC\n",
    "mv = None\n",
    "print(\"8. Memoryview reference set to None.\")\n",
    "\n",
